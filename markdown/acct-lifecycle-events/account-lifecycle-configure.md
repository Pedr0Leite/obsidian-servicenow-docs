# Configure the account onboarding playbook

Customize the [[account-lifecycle-account-onboard-playbook|account onboarding playbook]] to your organization's onboarding process by configuring its activities, tasks, data validation rules, and data import behavior.

The account [[account-lifecycle-onboard-playbook|onboarding playbook]] includes default activities, tasks, and data validation rules in the base system. Customize the playbook to match your organization's onboarding process by configuring its activities, tasks, data validation rules, and data import behavior.

-   **[[account-lifecycle-pad|Configure the onboarding playbook activities]]**  
Use Playbooks in Workflow Studio to configure the [[account-lifecycle-playbook-overview|account onboarding]] playbook activities.
-   **[[account-lifecycle-configure-simple|Configure onboarding tasks in playbook]]**  
You can [[account-lifecycle-config-cust-success|configure customer success]] onboarding tasks in a playbook using the Process Automation Designer.
-   **[[account-lifecycle-data-valid-assist|Configure data validation]]**  
Create field and record level validations in the Data validation assist table.
-   **[[account-lifecycle-import-builder|Configure the account onboarding data import task]]**  
Use the Import Builder to configure the [[account-lifecycle-use|account onboarding]] data import task.
-   **[[account-lifecycle-add-data-import|Add the data import task]]**  
Add the data import task that you’ve configured to the **Account lifecycle onboarding process** defined in the Process Automation Designer.
-   **[[account-lifecycle-setup-rec-action|Set up recommended actions for account onboarding]]**  
Set up recommended actions to display relevant recommendations in the account onboarding playbook.
-   **[[account-lifecycle-backup|Run scheduled job to back up staging table]]**  
You can take a backup of the data on the staging table at periodic intervals.
