# AI Agent Advisor in Now Assist Center

Use [[now-assist-center-ai-agent-advisor|AI Agent Advisor in Now Assist Center]] to automatically discover automation opportunities in your instance and deploy AI agents to implement the automations.

[[now-assist-center-landing-page|Now Assist Center]] enables you to perform the following activities using AI Agent Advisor:

-   [[now-assist-center-view-automation-opportunities|View your automation opportunities]]

    Review the automation opportunities that [[ai-agent-advisor-landing-page|AI Agent Advisor]] has identified for your instance.

-   [[activate-automation-opportunity-now-assist-center|Implement an automation opportunity from Now Assist Center]]

    Deploy a matched AI agent or a new agent to automate a resolution for an identified automation opportunity.

-   [[now-assist-center-edit-opportunity-agent|Edit an AI agent from an automation opportunity]]

    Edit an AI agent that was created for an automation opportunity.


For more information, see [[now-assist-center-using-ai-agent-advisor|Using AI Agent Advisor in Now Assist Center]].
