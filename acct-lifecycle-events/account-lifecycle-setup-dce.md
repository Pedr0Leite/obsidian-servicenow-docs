# Configure the Data context engine

Configure the [[account-lifecycle-setup-metric-data|Data context engine]] to collect data from different data sources and associate it with a relevant context table.

|Task|Description|
|----|-----------|
|[[account-lifecycle-define-data-source|Define the data source]]|Specify whether data is being collected from PA indicators, external sources, or calculated metrics.|
|[[account-lifecycle-define-context-engine-mapper|Configure the Context Engine Mapper]]|Associate the data source with an engagement, success outcome, or product usage table, and specify the record in the context table for which the data is applicable.|
|[[account-lifecycle-configure-dce-visual|Configure DCE Visualizations]]|Configure and display metrics based on context and product usage patterns.|
