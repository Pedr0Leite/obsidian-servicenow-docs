# Create a Health and Safety incident in Critical Event Management

Report a [[health-safety-overview|Health and Safety]] incident through the **Related records** Related Item in [[critical-event-management|Critical Event Management]].

## Before you begin

Role required: sn\_ohs\_im.incident\_reader, sn\_ohs\_im.incident\_writer, and sn\_ohs\_im.incident\_create

## Procedure

1.  Navigate to **Workspaces** &gt; **[[health-safety-core-workspace|Health and Safety Workspace]]** and select the **Critical Event Management** tab.

2.  Open the critical event.

3.  Select the **Related records** Related Item.

4.  Select **Health and Safety incidents**.

5.  If you want to populate any existing Health and Safety incidents, select the **Add** button, select the check box next to one or more incidents, and select **Add**.

6.  Create a Health and Safety incident by selecting **New**.

    The record opens in a new record tab.

7.  Complete the **Report an incident** playbook.
