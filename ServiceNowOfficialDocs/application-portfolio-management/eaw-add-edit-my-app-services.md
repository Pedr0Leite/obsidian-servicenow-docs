# Add or edit your application services

Add or edit an application service created by you in the [[ea-workspace|Enterprise Architecture Workspace]].

## Before you begin

Role required: sn\_apm.apm\_analyst

## Procedure

1.  Navigate to **Workspaces** &gt; **Enterprise Architecture Workspace**.

2.  Open the Portfolio List view by clicking the Portfolio icon \[Omitted image "portfolio-icon.png"\] Alt text: Portfolio icon.

3.  Select the expand row icon \(\[Omitted image "ExpandIcon.png"\] Alt text: Expand Row icon\) next to **My Entities**.

4.  Select **My Application Services**.

5.  Add or edit an application service.

    -   To add an application service, select **Add**.
    -   To update details of an existing application service, select the record then select **Edit**.
6.  On the form, fill in the fields.

    For field information, see [[eaw-app-service-form|Application service form]].

7.  Select **Save** or **Update**.
