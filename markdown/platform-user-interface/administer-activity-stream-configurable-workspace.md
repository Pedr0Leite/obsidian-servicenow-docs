# Administering Activity stream for Configurable Workspace

Administer [[activity-stream-configurable-workspace|Activity stream]] features that aren't configured in UI Builder with components.

-   **[[configure-activity-stream-general|Configure stacked view]]**

    [[configure-onboarding-modals|Configure]] a stacked view option for journal fields in the Activity stream.

-   **[[config-activity-stream-rte|Configure rich text editor]]**

    Configure a rich text editor for journal fields in the Activity stream.

-   **[[disable-mandatory-field-post-button|Configure Post button]]**

    Configure the **Post** button in the Activity stream to display and function continuously by using the **glide.activity.[[activity-stream-compose-configurable-workspace|compose]].can\_post\_mandatory\_fields** system property.

-   **[[configure-activity-stream-avatars-icons|Configure tile avatars]]**

    Configure Activity stream tiles to display avatars by using the **glide.activity.show\_tile\_icons** system property.

-   **[[config-journal-field-icons-and-colors|Configure tile icons]]**

    Configure custom icons for tiles in the Activity stream.

-   **[[tags-activity-stream-admin|Create tags]]**

    Configure custom [[c_Tags|tags]] to filter through entries in the Activity stream.

-   **[[configure-role-based-access-control-for-creating-personalized-filter-sets|Configure filter set access]]**

    Configure access to create filter sets in the Activity stream by restricting specified users.

-   **[[configure-last-viewed-timestamps-activity-stream|Configure most recent reply]]**

    Configure the Activity stream to reduce visual clutter by displaying the most recent reply first.
