# Impact Value Management Data Collection for APM

Install and configure the Data Collection Content Pack for APM.

Perform the steps in the following order:

1.  [[dc-apm-install|Install the Data Collection Pack for APM]]
2.  [[dc-apm-config|Configure Data Collection for APM]]
3.  [[dc-apm-activate|Activate Data Collection for APM]]
4.  [[dc-apm-installation-artifacts|Review APM artifacts]]

You can also modify the configuration at any point, and maintain customizations when you are ready to upgrade. For more information, see [[dc-all-modify|Modifying the Data Collection configuration]] and [[dc-all-upgrade|Upgrading the Data Collection Content Pack]].
