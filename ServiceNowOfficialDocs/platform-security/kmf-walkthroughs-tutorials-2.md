# Column Level Encryption Enterprise examples

These examples walk you through the [[encryption-landing|encryption]] of fields and attachments using customer-supplied keys.

-   **[[column-level-encryption-enterprise-walkthrough-2|Column Level Encryption Enterprise walkthrough]]**  
This walkthrough shows you how to encrypt a field in your instance [[using-column-level-encryption|using Field Encryption]] Enterprise with the [[encryption|Key Management Framework]] \(KMF\). It also shows you how to use your own key.
-   **[[attachment-encryption-walkthrough|Attachment encryption walkthrough]]**  
This walkthrough shows you how to encrypt an attachment in your instance using Field Encryption Enterprise with the Key Management Framework \(KMF\). It also shows you how to use your own key.
