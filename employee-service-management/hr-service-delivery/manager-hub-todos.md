# Default configurations for To do's

Default set of To do's configurations are available to use after you have activated [[employee-center-landing-page|Employee Center]] Core. These configurations display managerial [[emp-slate-inbox|tasks and requests]] in [[manager-hub-overview|Manager Hub]].

To dos filters is a module within Employee Center. For more details on how to configure the filters, see [Create or modify a to-dos filter](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/employee-experience-foundation/configure-to-dos-page-filters.md)

|Name|Description|
|----|-----------|
|Approvals|Filters to display requests that require approval of a manager.|
|My team tasks|Filter to display [[jny-team-tasks|team tasks]] to a manager.|

The following filter conditions are also available: HR approvals, HR tasks, HR tasks for UR, [[journey-accelerator-1|Journey accelerator]] to-dos, Journey accelerator to-dos- Adhoc, Request approvals, [[universal-task-landing|Universal task]], and UR approvals.

For more information on the filters, navigate to **Employee Center** &gt; **Administration** &gt; **To-dos configuration** &gt; **To-dos filters**.

For more information on the filter conditions, navigate to **Employee Center** &gt; **Administration** &gt; **To-dos configuration** &gt; **To-dos config filter conditions**.
