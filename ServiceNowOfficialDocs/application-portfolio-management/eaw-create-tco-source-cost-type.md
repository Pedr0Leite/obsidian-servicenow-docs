# Create a TCO source cost type

Create a source cost type for the TCO source. Specify the type of cost being ingested in [[ea-workspace|Enterprise Architecture Workspace]].

## Before you begin

Role required: sn\_apm.apm\_analyst

## About this task

After a source is created for the TCO, you can create a source cost type from the related list.

## Procedure

1.  Navigate to **Workspaces** &gt; **Enterprise Architecture Workspace** &gt; **Setup**.

2.  Select **TCO**.

3.  Select **Sources**.

4.  Select a source to open it.

5.  Select the **TCO Source Cost Types** tab.

6.  Select **New**.

7.  Enter a name and domain for the source cost type.

    The **Source** is field is auto populated with a source name.

8.  Select **Save**.
