# Now Assist for Platform Health

[[impact-landing-page|Impact]]'s [[platform-health-idi|Platform Health]] suite comes with Now Assist powered features designed to [[io-help|help]] customers proactively identify, prioritize, and resolve technical debt.

These tools leverage AI to accelerate leading practice adoption and optimize platform health across your ServiceNow instances.

\[Omitted image "now-assist-platform-health.png"\] Alt text: Overview of how Now Assist for Platform Health prevents and addresses technical debt.

## [[real-time-prevention-monitoring-combined|Real-time prevention monitoring for Scan Engine]]

Real-time prevention monitoring functionality delivers AI-generated code fixes at the point of development, as code is written. Leading practice violations are flagged so developers can instantly generate, review, and apply AI-recommended fixes. Cleaning the code during development prevents technical debt before it releases to production.

## [[code-fix-ai-agent-scan-engine|Fix code in real-time with Now Assist]]

Resolve Scan Engine findings after scans to accelerate resolution of existing technical debt across your instance. For findings already identified by the Scan Engine, developers can generate AI-recommended fixes in bulk, review proposed changes, and apply fixes. A backlog of violations become clean code and resolved issues without timely manual development remediation.
