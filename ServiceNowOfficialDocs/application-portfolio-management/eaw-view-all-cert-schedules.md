# View all certification policies

View the list of all certification policies and manage them in the [[ea-workspace|Enterprise Architecture Workspace]].

## Before you begin

Role required: certification\_admin

## Procedure

1.  Navigate to **Workspaces** &gt; **Enterprise Architecture Workspace**.

2.  Open the Setup page by selecting the Setup icon \(\[Omitted image "eaw-icon-setup.png"\] Alt text: setup icon.\).

3.  Select the expand row icon \(\[Omitted image "ExpandIcon.png"\] Alt text: expand row icon.\) next to **Certification Policies**.

4.  Select **All**.
