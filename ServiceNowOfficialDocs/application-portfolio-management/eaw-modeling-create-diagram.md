# Create a blank diagram using modeling in the EA Workspace

Create a diagram and synchronize it to the ServiceNow database.

## Before you begin

Role required: sn\_apm.apm\_user

## Procedure

1.  Navigate to **Workspaces** &gt; **[[ea-workspace|Enterprise Architecture Workspace]]**.

2.  Open the Modeling page by selecting the Modeling icon \(\[Omitted image "icon-modeling-logo.png"\] Alt text: Modeling\).

3.  Select **New**.

4.  Select **Blank diagram**.

5.  Enter a name for the artifact and select the architectural category it’s associated with.

6.  Select **Create diagram**.


## Result

An empty diagram page opens in a new tab. You can add shapes and create a diagram.

**Note:**

-   You can add a shape to the canvas by selecting the shape or by dragging it from the **Shapes** palette to the canvas.
-   Before adding a shape, hover over its icon in the **Shapes** panel to see a preview popover showing a larger view of the shape and its label.
