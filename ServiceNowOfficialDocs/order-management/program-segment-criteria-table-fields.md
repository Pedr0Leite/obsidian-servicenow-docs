# Program segment criteria table fields

You can establish a mapping between the program segment mapping table and the criteria using the program segment criteria \(sn\_prm\_program\_segment\_criteria\) table.

|Field|Type|Description|
|-----|----|-----------|
|Program Segment Mapping|Reference|Reference to the records in the program segment mapping \(sn\_prm\_program\_segment\_mapping\) table|
|Criteria|Reference|Reference to the entity criteria \(sn\_req\_criteria\_customer\_condition\) table|
|Active|True/False|Current state of the program segment criteria, whether active or not|
