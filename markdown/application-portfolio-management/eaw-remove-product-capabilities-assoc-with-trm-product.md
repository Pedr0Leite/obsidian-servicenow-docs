# Remove product capabilities associated with a Technology Reference Model product

You can remove the product capabilities associated with Technology Reference Model \(TRM\) products in the [[ea-workspace|Enterprise Architecture Workspace]], confirming only the relevant and current product capabilities are associated with the TRM products.

## Before you begin

Role required: sn\_apm.apm\_user

## Procedure

1.  Navigate to **Workspace** &gt; **Enterprise Architecture Workspace**.

2.  Open the Technology Portfolio page by selecting the Technology Portfolio icon \[Omitted image "technology-portfolio-icon.png"\] Alt text: Technology portfolio icon.

3.  In the **TRM catalog** tab, select **TRM product list** from the View list.\[Omitted image "trm-product-view-dropdown.png"\] Alt text: TRM view list drop-down list.

4.  Select the expand row icon \(\[Omitted image "ExpandIcon.png"\] Alt text: Expand Row icon\) next to the TRM product that you want to remove the product capabilities for.

5.  Select the TRM product.

6.  Select **Product capabilities**.

7.  Select the check box next to the product capability that you want to remove and select **Remove**.

    A confirmation message appears.

8.  Select **Remove**.


## Result

The product capability is removed from the TRM product.
