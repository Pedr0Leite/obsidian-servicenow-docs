# How does Task Intelligence Work?

## 

 Intelligence starts by predicting the language and sentiment of newly 
created cases. Also, it can determine if the case is a duplicate or 
spam. Next, Task Intelligence increases self-solve by automatically 
responding with knowledge articles or recommending case resolutions. 
Then, for cases needing agents, Task Intelligence categorizes the case 
and extracts values from attached documents (if using DocIntel) to 
populate relevant fields. Finally, as agents work toward a resolution, 
Task Intelligence assists them with recommended knowledge articles, 
similar resolved cases, real-time recommendations, summarizing 
hand-offs, and wrapups.
