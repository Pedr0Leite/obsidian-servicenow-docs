# Configure Hiring

Set up the [[hiring-dashboard|Hiring tab]] in the [[manager-hub-overview|Manager Hub]] to empower the [[hiring-tab-concept|hiring]] managers to make quality hires. Create a seamless process and easy communication with the recruiter.

Set up the [[rfy-ec|Recommended for you]] section to add and [[view-hiring-content|view hiring content]] easily. For more information on setting up Recommended for you, see [Recommended for you](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/employee-experience-foundation/rfy-ec.md).

Enable Now Assist Q&amp;A Genius Results to generate answers from articles most relevant to the search query. For more information, see [Enabling Now Assist Q&amp;A Genius Results](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-administration/enabling-now-assist-qa-grs.md).

Customize the [[new-job-form|job requisition form]] fields according to your organizational requirements. For more information, see [Create or edit a catalog item](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/servicenow-platform/t_DefineACatalogItem.md).
