# Certificate based authentication

[[certificate-based-authentication|Certificate-based authentication]] lets you mutually authenticate inbound API requests using [[c_Certificates|certificates]] from a trusted Certificate Authority \(CA\).

-   **Certificate-based authentication for Inbound web services**

    Authenticate inbound requests to ServiceNow SOAP and REST APIs. To set up mutual [[c_Authentication|authentication]] for inbound web services, see [Set up Certificate-based authentication](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-security/certificate-based-authentication/set-up-mutual-auth.md).
