# Create a business application and associate it with a Technology Reference Model product

You can create business applications and associate it with Technology Reference Model \(TRM\) products in the [[ea-workspace|Enterprise Architecture Workspace]] \(EA Workspace\).

## Before you begin

Role required: sn\_apm.apm\_analyst

## Procedure

1.  Navigate to **Workspace** &gt; **Enterprise Architecture Workspace**.

2.  Open the Technology Portfolio page by selecting the Technology Portfolio icon \[Omitted image "technology-portfolio-icon.png"\] Alt text: Technology portfolio icon.

3.  In the **TRM catalog** tab, select **TRM product list** from the View list.\[Omitted image "trm-product-view-dropdown.png"\] Alt text: TRM view list drop-down list.

4.  Select the expand row icon \(\[Omitted image "ExpandIcon.png"\] Alt text: Expand Row icon\) next to the TRM product that you want to create and associate new business applications to.

5.  Select the TRM product.

6.  Select **Business Applications**.

7.  To create a business application and associate with the TRM product, select **New**.

8.  In the Create New Business Application page, fill in the form fields.

    For field information, see [[eaw-business-application-form|Business application form]].

9.  Select **Save**.


## Result

The business application is added to the TRM product.
