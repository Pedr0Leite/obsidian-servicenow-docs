# Threat Lookup

Scan selected [[c_Observables|observables]] for [[threat-intelligence-malware|malware]] using VirusTotal and CrowdStrike Falcon Intelligence. This workflow checks observables against both [[threat-intel-landing-page|threat intelligence]] sources and returns detailed security analysis results.

-   **[TISC VirusTotal integration](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/security-management/threat-intelligence-security-center/tisc-virustotal-integration_0.md)**  
Request analysis of suspicious IP addresses, file hashes, and URLs through [[virustotal-landing-page|VirusTotal integration]] to determine if they are malicious during security investigations.
-   **[TISC CrowdStrike Falcon Intelligence integration](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/security-management/threat-intelligence-security-center/tisc-crowdstrike-falcon-intelligence-integration.md)**  
[[integrating-threat-intelligence-security-center|Integrate]] CrowdStrike Falcon Intelligence with [[security-operations-landing-page|Security Operations]] to enhance threat detection and response capabilities with real-time cyber security intelligence.
