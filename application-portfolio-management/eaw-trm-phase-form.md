# TRM Phase form

Define your own TRM phase or edit an existing TRM phase.

## TRM Phase form fields

|Field|Description|
|-----|-----------|
|Name|Name of the phase.|
|Description|Description about the phase.|
|Production Approved|Option to mark as approved for production or not.|
|Color|Color that defines the TRM phase. Select a color from the drop-down list.|
|Shape|Shape that defines the TRM phase. Select a shape from the drop-down list.|
