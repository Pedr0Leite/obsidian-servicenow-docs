# Configuring Hiring Experiences

Install and configure the [[hiring-experiences-landing|Hiring Experiences]] suite for optimal usage according to your organizational requirement.

## Configuration overview

-   [Plan your installation](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/hr-service-delivery/plan-installation-ta.md)

    Know the required plugins and the dependencies to configure your Hiring Experiences suite optimally.

-   [Install Hiring Connector for third party integration](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/hr-service-delivery/install-hiring-connector.md)

    Install the [[hiring-tab-concept|Hiring]] Connector application to integrate third party applications with your Hiring Experiences suite.

-   [Modify purge policies to clean up data](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/hr-service-delivery/set-up-purge-pollicies-ta.md)

    Modify the default data purge policies and create rules to archive and delete data for applications in the Hiring Experiences suite, as needed.

-   [Configuring Hiring](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/hr-service-delivery/configuring-hiring-tab.md)

    Set up the [[hiring-dashboard|Hiring tab]] application within the [[manager-hub-overview|Manager Hub]] to provide a seamless hiring workspace for hiring managers.

-   [Configuring Recruitment Workspace](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/recruitment-workspace/config-recruitment.md)

    Set up the [[recruitment-workspace-concept|Recruitment workspace]] and create an integrated workspace and a collaborative communication channel between recruiters and the other hiring personas.

-   [Configuring Interview Management](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/interview-management/configuring-interview-management.md)

    Set up the [[interview-management-landing|Interview management]] application to get started with your interview scheduling and management journey.

-   [Configuring Applicant Center](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/applicant-center/config-applicant-center.md)

    Set up the [[applicant-center-concept|Applicant Center]] for your applicants to have a personalized portal for all their queries and communication.

-   [Configuring Talent profile](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/hr-service-delivery/config-tp.md)

    Set up the [[talent-profile-landing|Talent profile]] to attract and manage top talent from the industry.

-   [Install and configure Opportunity Marketplace](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/opportunity-marketplace/egd-oppt-mrktplc-inst-config.md)

    Set up the [[egd-oppt-mrktplc-landing|Opportunity Marketplace]] application.
