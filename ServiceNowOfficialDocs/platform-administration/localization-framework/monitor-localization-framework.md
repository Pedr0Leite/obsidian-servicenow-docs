# Monitor the Localization Framework

Track [[ia-localization-il|localization]] performance and [[language-picker-ui|request translations]] with [[localization-framework-landing|Localization Framework]] Dashboards.

-   **[Localization Insights dashboard](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-administration/localization-framework/localization-insights-dashboard.md)**  
The [[localization-insights-dashboard|Localization Insights dashboard]] provides reports on translations, grouped into tabs by artifact type. The reports are generated by [[c_ScheduledJobs|scheduled jobs]] based on Localization Insights records.
-   **[Request translations from Insights Dashboard](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-administration/localization-framework/request-translations-insights-dashboard.md)**  
[[request-translations-multiple-items|Request translations for Service Catalog items]] or Virtual Agent topics from the insights dashboard to localize them into the selected language based on their translation status.
