# Lists and forms in scoped applications

The current [[c_ApplicationContext|application context]] determines what customization and form design options are available when working with lists and forms in scoped applications.

The user interface uses visual indicators to identify a list or form in the same or different [[c_ApplicationScope|application scope]].
