# Add a related entity to an architectural document

Associate an architectural artifact to existing entities such as business capabilities, business applications, digital integrations, and digital interfaces. The association creates a relationship between the artifact and related entities.

## Before you begin

Role required: sn\_apm.apm\_user

## Procedure

1.  Navigate to **Workspace** &gt; **[[ea-workspace|Enterprise Architecture Workspace]]**.

2.  Open the Portfolio List view by selecting the Portfolio icon \[Omitted image "portfolio-icon.png"\] Alt text: Portfolio icon.

3.  Select the expand row icon \(\[Omitted image "ExpandIcon.png"\] Alt text: Expand Row icon\) next to **Information Portfolio**.

4.  Select **Architectural Documents**.

5.  Select the architectural document that you want to add a related entity to.

    A new page appears and the details of the architectural document are displayed.

6.  Select the **Related Entities** tab.

7.  Select **New**.

8.  On the form, fill in the fields.

    For a description of the field values, see [[eaw-artifact-releated-entity-form|Create related entities form]].

9.  Select **Save**.
