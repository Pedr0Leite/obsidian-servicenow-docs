# Use AI to build apps faster

Learn about the suite of AI tools available on the ServiceNow AI Platform and how you can use them to build applications more efficiently.

When developing applications on the ServiceNow AI Platform, you can use AI to help speed up the development process. ServiceNow has a suite of [[dev-get-start-ai-assets|AI assets]] and products that can enable you to develop applications and app metadata more efficiently. Whether you're looking to expedite individual development tasks, or build entire applications through vibe coding, ServiceNow has the AI tools that you need to supercharge your development.

-   **[[dev-get-start-contextual-assistance|Contextual assistance]]**  
[[now-assist-for-creator-landing|Now Assist for Creator]] contains AI assets that provide contextual assistance for handling development tasks more efficiently.
-   **[[dev-get-start-vibe-coding|Vibe coding]]**  
Vibe coding is an AI-driven approach to app development in which you build applications through conversations with AI. On the ServiceNow AI Platform, developers can [[use-build-agent|use Build Agent]] to vibe code entire applications.
-   **[[dev-get-start-ai-development-best-practices|General guidelines for AI-assisted development]]**  
When using AI-assisted tools to build or vibe code applications and app components on the ServiceNow AI Platform, the quality of the output depends on the quality of your instructions. Use these guidelines to write effective instructions and improve the products of your conversations with AI tools.
-   **[[dev-get-start-choosing-your-build-approach|Choosing your build approach]]**  
Understand when to use AI-assisted tools like the [[sns-now-assist-app-gen-landing|app generation]] skill with Now Assist for Creator and [[build-agent|Build Agent]] versus when to build an application manually on the ServiceNow AI Platform.
