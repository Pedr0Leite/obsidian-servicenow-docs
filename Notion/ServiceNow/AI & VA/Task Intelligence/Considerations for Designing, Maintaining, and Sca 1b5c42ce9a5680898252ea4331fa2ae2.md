# Considerations for Designing, Maintaining, and Scaling the Solution

## Maintaining Performance

To maintain performance, ServiceNow recommends the following steps:

1. Periodically retrain the models on a new set of historical data
2. Using the Admin Console, compare these results of the old model versus the model trained on the new set of historical data
3. Select the better-performing model
