# View related details for an affected user artifact

If your case includes affected user artifacts, you can view any related details contained in each affected user referenced by the case.

## Before you begin

-   The [[threat-intel-landing-page|Threat Intelligence]] plugin must be activated to use [[case-mgmt|Security Case Management]].

Role required: sn\_ti.case\_user\_write

## Procedure

1.  Open a case that contains affected user artifacts.

2.  Click the **Users** tab.

3.  Click Related Details for the affected user artifact you want to view related details for.

    \[Omitted image "related-details-user.png"\] Alt text: View related deatils for a user

4.  Click any of the tabs to view specific types of details for the selected artifact.

5.  You can also click **Security Annotation** tab to view annotations for the selected artifact.
