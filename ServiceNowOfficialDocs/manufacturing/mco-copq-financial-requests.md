# Create a CoPQ financial request

Create a CoPQ financial request for remediation action plan.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **Workspaces** &gt; **CSM/FSM Configurable Workspace** &gt; **List** &gt; **Remediation Action Plan** &gt; **CoPQ Financial Request**.

2.  Select **New**.

3.  On the CoPQ Financial Request form, fill in the fields.

    For a description of the field values, see [[mco-copq-financial-requests-form|CoPQ financial requests form]].

4.  Select **Save**.
