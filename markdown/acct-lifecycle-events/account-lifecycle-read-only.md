# Read-only field enhancements

Read-only field protections have been moved from the client side to the server side. This prevents users from manipulating data through client-side methods.

Starting with [[account-lifecycle-events-landing|Customer Success Management]] 5.3.11, the following plugins have security enhancements for read-only fields:

-   [[account-lifecycle-events-customer-success-about|Customer Success]] Management \[com.sn\_acct\_lc\]
-   Touchpoint Meetings \[com.sn\_meeting\_mgmt\]
-   Technology Core \[com.sn\_ti\_core\]
-   [[account-lifecycle-setup-metric-data|Data Context Engine]] \[com.sn\_data\_ctx\_engine\]
-   Telecom Core \[com.sn\_ind\_tsm\_core\]
-   Product Capability Core \[com.sn\_prod\_cap\_core\]
