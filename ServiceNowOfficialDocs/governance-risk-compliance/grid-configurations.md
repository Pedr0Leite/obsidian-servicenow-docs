# Grid configurations and categories

You can set up the grid configuration to render the BIA dependency assessment grid with configured columns. You can define the grid category such as Dependency Assessment for which the grid configuration is applicable.

The example shows the grid configuration in the [[business-continuity-mangmt-overview|Business Continuity Management]] application. \[Omitted image "new-grid-configurations.png"\] Alt text: Grid configuration.

**Note:** Grid configuration is used in BCM classic Workspace only.

For more information on how to configure the grid in the Business Continuity Management application, see [[setup-bcm-grid-config|Set up grid configuration for BIA dependency assessment]].

The example shows the configuration of a grid category in the Business Continuity Management application. \[Omitted image "new-grid-category.png"\] Alt text: Configuration of a grid category.

**Note:** Grid categories are used in BCM classic Workspace only.
