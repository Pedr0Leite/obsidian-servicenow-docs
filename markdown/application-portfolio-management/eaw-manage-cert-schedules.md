# Add or edit a certification policy

Add or edit a certification policy in the [[ea-workspace|Enterprise Architecture Workspace]].

## Before you begin

Role required: certification\_admin

## Procedure

1.  Navigate to **Workspaces** &gt; **Enterprise Architecture Workspace**.

2.  Open the Setup page by selecting the Setup icon \(\[Omitted image "eaw-icon-setup.png"\] Alt text: setup icon.\).

3.  Select the expand row icon \(\[Omitted image "ExpandIcon.png"\] Alt text: expand row icon.\) next to **Certification Policies**.

4.  Select **All**.

5.  Create or update a CMDB Data Management Certification Policy.

    -   To [[eaw-create-policy|create a certification policy]], select **New**.
    -   To update an existing certification policy, select a certification policy, then select **Edit**.
6.  On the CMDB Data Management Certification Policy form, fill in the fields.

    For field information, see [[eaw-certification-policy-form|Certification policy form in Enterprise Architecture Workspace]].

7.  Select **Save**.
