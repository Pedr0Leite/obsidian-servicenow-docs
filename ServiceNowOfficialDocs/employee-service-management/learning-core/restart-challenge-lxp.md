# Restart a learning challenge

Restart a challenge when you want to reset the progress you have made on the challenge.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **[[employee-center-landing-page|Employee Center]]** &gt; **[[overview-lxp|My Learning]]**.

2.  On the [[exploring-learning-exp|learning]] challenge widget, select the context \[Omitted image "icon-menu.png"\] Alt text: context icon icon.

    **Note:** You might see different labels on the widget depending on your current status on the challenge. If you have started the challenge, you will see Your current challenge.

3.  Select **Start again**.

4.  On the **Do you want to restart this challenge** pop-up, click **Yes, I'm done**.
