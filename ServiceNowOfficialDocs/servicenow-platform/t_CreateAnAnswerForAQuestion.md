# Create answers for questions

Create answer options for questions with the Choice or Likert Scale data type.

## Before you begin

Role required: none

## About this task

When you create a question with one of these data types, the Assessment Metric Definitions related list appears. Each metric definition appears as one answer option for a question on a quiz. For example, the question **What type of men's trousers are not permitted by company policy?** might have these three answers configured as metric definitions: **Slacks**, **Jeans**, and **Casual cotton**.

## Procedure

1.  Open the question record you want to edit.

2.  Click **New** in the **Assessment Metric Definitions** related list.

3.  Set the order in which the answers appear in the quiz by providing a hierarchical number in the **Value** field.
