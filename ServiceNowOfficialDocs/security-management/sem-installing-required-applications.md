# Installing Security Exposure Management Workspace applications

The required applications for the [[sem-workspace-user-interface|Security Exposure Management Workspace]] are available from the ServiceNow Store.

1.  See [Exploring the ServiceNow Store](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-administration/exploring-servicenow-store.md) for more information about applications, integrations, and products in the ServiceNow® Store.
2.  See [[sem-install-prerequisites|Migration upgrade prerequisites]] and [[sem-migration-prereq-reference-data|Migration upgrade reference information]] for more information about setup requirements for the USEM migration.
3.  See [[sem-install-required-applications|Download and activate applications]] for a list of the required applications and installation steps for the Security Exposure Management Workspace.
4.  See [[sem-components-installed|Security Exposure Management Workspace Components]] for roles and tables that are installed with the applications.
