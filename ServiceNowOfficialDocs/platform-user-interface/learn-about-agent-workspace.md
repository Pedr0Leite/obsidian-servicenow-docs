# Exploring Configurable Workspace

Get an overview of Configurable Workspace so you can understand how to [[configure-onboarding-modals|configure]] and use it for your organization.

## Configurable Workspace overview

A [[configurable-workspace-glossary|workspace]] provides a suite of tools in a single, focused work area that enables an agent to complete an entire job efficiently.

For example, an agent can receive an incoming case, review its details, communicate with the requester, and close the case within a single interface.

Configurable Workspace is a workspace [app shell](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/configurable-workspace-glossary.md) provided by the ServiceNow AI Platform®.

ServiceNow offers prebuilt Configurable Workspace experiences to target specific users and issues. For a list of workspaces ServiceNow offers, see [[list-of-workspaces|List of workspaces]].

## UI Builder and Configurable Workspace

[Admins](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/configurable-workspace-glossary.md) use [UI Builder](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/configurable-workspace-glossary.md) to [[create-configurable-workspace-experience-uib|create a Configurable Workspace experience]] from scratch or to modify prebuilt workspaces to match their workflows, branding, and data needs.

UI Builder is a WYSIWYG web user interface builder that uses Next Experience Components as the building blocks for Configurable Workspace [pages](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/configurable-workspace-glossary.md). Each [component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/configurable-workspace-glossary.md) provides a specific function like showing a list of records or displaying a form.

## Parts within a Configurable Workspace

Configurable Workspace experiences are made up of several parts that help agents navigate, manage, and work on records.

-   **[[lists-configurable-workspace|Lists]]**

    Lists display multiple records in a Configurable Workspace, such as cases, task, or incidents, so that agents can find and work on records quickly.

-   **[[form-configurable-workspace|Forms]]**

    Forms enable agents to view, enter, and update details about the record.

-   **[[activity-stream-configurable-workspace|Activity stream]]**

    The Activity stream helps agents stay informed by displaying a running history of updates and communications within a record.

-   **[[activity-stream-compose-configurable-workspace|Compose]]**

    The Compose area in a record page enables agents to communicate with customers and other agents through comments, work notes, and emails.

-   **[[contextual-sidebar-configurable-workspace|Tabs sidebar]]**

    The tabs sidebar is a flexible panel on a record page that provides agents with additional tools and information displayed as tabs like [[agent-assist-configurable-workspace|Agent assist]] and [[form-templates-configurable-workspace|form templates]].
