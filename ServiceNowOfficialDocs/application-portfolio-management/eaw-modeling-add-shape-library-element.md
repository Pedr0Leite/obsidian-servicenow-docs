# Add a shape library element

Create a shape library element to associate a shape and its diagram action to the shape library.

## Before you begin

Role required: sn\_apm.apm\_admin

## Procedure

1.  Navigate to **Workspaces** &gt; **[[ea-workspace|Enterprise Architecture Workspace]]**.

2.  Open the Setup page by selecting the Setup icon \(\[Omitted image "eaw-icon-setup.png"\] Alt text: setup icon.\).

3.  Select the expand row icon \(\[Omitted image "ExpandIcon.png"\] Alt text: expand row icon.\) next to **Enterprise Modeling and Visualization**.

4.  Select **Shape Libraries**.

5.  Open the shape library for which you want to associate a shape element and its diagram action.

6.  Select the Shape Library Elements tab.

7.  Select **New**

8.  On the Shape Library Element form, fill in the fields.

    For field information, see [[eaw-modeling-shape-library-element-form|Shape library element form]].

9.  Select **Save**.
