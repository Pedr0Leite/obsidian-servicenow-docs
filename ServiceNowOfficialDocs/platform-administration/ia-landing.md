# Setup Hub

The Setup Hub provides a unified, guided experience to set up, configure, and adopt ServiceNow products. It connects Admin Home, product hubs, and the Configuration Console into a single path from product activation to full configuration.

Administrators can manage entitlements, install required components, apply recommended [[clone-configurations-tab|configurations]], complete guided setup steps, track progress, and package changes for promotion. The experience reduces manual effort and complexity by streamlining installation and configuration tasks. The Setup Hub is available with or without AI. All setup capabilities—including installation, default configurations, guided steps, and update set packaging are supported without AI, while AI-assisted features, such as agents and conversational experiences, are available when included in the subscription.
