# Universal Task reference

You can refer to the [[reference-hr-service-delivery-advanced-integration-with-workday|reference]] topics to learn more about the universal tasks, including lists and forms.

-   **[Universal Task form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/universal-task/universal-task-form.md)**  
[[universal-task-form|Universal Task form]] to create a task record.
-   **[Universal Task Configuration form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/universal-task/unitask-config-form.md)**  
Use the [[unitask-config-form|Universal Task Configuration form]] to associate [[universal-task-landing|Universal Task]] to your service.
-   **[Employee form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/universal-task/ut-emp-form.md)**  
Use the [[ut-emp-form|employee form]] to create a new employee form record for Universal Task
-   **[Employee Input Task Configuration form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/universal-task/ut-empform-config-form.md)**  
Use the [[ut-empform-config-form|Employee Input Task configuration form]] to configure the roles that can view completed Universal Task employee form.
-   **[Universal Task template form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/universal-task/ut-task-template-form.md)**  
Use the [[ut-task-template-form|Universal Task template form]] to [[create-unitask|create a Universal Task]] template.
