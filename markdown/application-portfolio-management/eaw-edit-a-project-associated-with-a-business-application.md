# Edit a project associated with a business application

Use application rationalization to edit existing projects associated with business applications.

## Before you begin

You can make updates to the demand record directly from the list without leaving the list view.

Role required: sn\_apm.apm\_analyst

## Procedure

1.  Navigate to **Workspaces** &gt; **[[ea-workspace|Enterprise Architecture Workspace]]**.

2.  Open the Application Rationalization page by selecting the application rationalization icon \(\[Omitted image "icon-app-rationalization.png"\] Alt text: Application rationalization icon.\).

3.  Select the expand row icon \(\[Omitted image "ExpandIcon.png"\] Alt text: Expand row icon.\) next to the business application that you want to edit the associated project details for.

    A list of projects associated with that business application is displayed.

4.  Select the project that you want to edit.

    A new pane appears on the List view screen and the details of the project are displayed.

    **Note:** Select **Full details** to view more details about the project.

5.  On the Edit a project form, fill in the fields.

    For a description of the field values, see [[eaw-edit-project-form|Edit a project form]].

6.  Select **Update**.
