# Create a sub-capability

Add a sub-capability to the capability hierarchy map.

## Before you begin

Role required: sn\_apm.apm\_user

## Procedure

1.  Navigate to **Workspaces** &gt; **[[ea-workspace|Enterprise Architecture Workspace]]**.

2.  Open the Business Capability Hierarchy page by selecting the Business Portfolio icon \(\[Omitted image "business-portfolio-icon.png"\] Alt text: Business Portfolio icon.\).

3.  Select the row context menu icon \[Omitted image "eaw-icon-menu.png"\] Alt text: and select **Create sub-capability**.

4.  On the form, fill in the fields.

    For a description of the field values, see [[eaw-create-a-sub-capability|Create a sub-capability form]].

5.  Select **Create**.
