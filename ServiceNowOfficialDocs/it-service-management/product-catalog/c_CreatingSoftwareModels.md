# Software models

Software models are created as part of the [[c_AssetManagement|asset management]] process. You can create product [[c_Models|models]] for software from the [[c_ProductCatalog|product catalog]], but you cannot administer all aspects of the software models.

Software models are used in software counters. For more information about creating software model, see [Create software models](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-asset-management/record-terms-software-licenses.md).
