# Using MITRE-ATT&amp;CK to detect and analyze threats

Use the MITRE-ATT&amp;CK framework across the [[threat-intel-landing-page|Threat Intelligence]] and the SIR module to detect and analyze threats to your organization.

-   **[[associate-mitre-with-sir|Associate MITRE-ATT&amp;CK information with security incidents]]**  
Associate the MITRE-ATT&amp;CK tactics and techniques to the security incident for better security incident and threat analysis.
-   **[[associate-mitre-observables|Associate MITRE-ATT&amp;CK information with observables]]**  
Associate MITRE-ATT&amp;CK tactics and techniques to an observable for better security incident and threat analysis at a granular level.
-   **[[associate-mitre-att-ck-information-with-security-case-management|Associate MITRE-ATT&amp;CK information with security case]]**  
Associate MITRE-ATT&amp;CK tactics and techniques to a security case for better [[case-mgmt|security case management]] and threat analysis at a granular level.
-   **[[rollup-threat-lookup-results|Rollup MITRE-ATT&amp;CK information using Threat Lookup results]]**  
If you have not enabled automatic rollup of MITRE-ATT&amp;CK information, you can do this manually.
-   **[[rollup-mitre-attack-information-detection-rules|Rollup MITRE-ATT&amp;CK information from detection rules]]**  
Enable rollup of MITRE-ATT&amp;CK information from the detection rules to the security incidents for better security incident and threat analysis.
-   **[[rollup-mitre-att-ck-information-from-child-security-incidents|Rollup MITRE-ATT&amp;CK information from child security incidents]]**  
If you have not enabled automatic rollup of MITRE-ATT&amp;CK information, you can do this manually.
-   **[[link-analysis-threat-hunt-mitre|Perform link analysis and threat hunting using MITRE-ATT&amp;CK specific filters]]**  
Correlate and perform link analysis of [[c_Observables|observables]], security incidents, and MITRE-ATT&amp;CK related information so that your organization can start hunting for threats.
-   **[[mitre-att-ck-heatmap-and-navigator|MITRE-ATT&amp;CK heat map and navigator]]**  
You can use the MITRE-ATT&amp;CK heat map and navigator for basic navigation and to visualize your overall technique detection coverage.
-   **[[mitre-dashboards|Using the MITRE-ATT&amp;CK dashboard]]**  
The MITRE-ATT&amp;CK dashboard provides an executive view of the data source coverage, tactics, and techniques that are used in your organization.
