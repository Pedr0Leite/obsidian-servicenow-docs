# Affected users in cases

You can create a new case from one or more affected users in the User \[sys\_user\] table. You can also add users to existing cases.

-   **[[create-cases-from-affected-users|Create a case from affected users]]**  
You can create a security case from affected users in the User \[sys\_user\] table. After the affected users have been used to create a new case, you can use [[case-mgmt|Security Case Management]] to analyze the data.
-   **[[add-users-to-cases-sir|Add affected users to existing cases]]**  
You can add affected users to one or more existing cases. After the user records have been added to cases, you can use Security Case Management to analyze the data.
