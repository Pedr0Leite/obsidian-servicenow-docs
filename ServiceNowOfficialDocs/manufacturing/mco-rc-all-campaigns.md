# View All campaigns

View a [[mco-rcl-clms|recall campaign]] using the All campaigns form.

## Before you begin

Role required: admin

## Procedure

1.  **Workspaces** &gt; **CSM/FSM Configurable Workspace** &gt; **Lists** &gt; **Recall Management** &gt; **All Campaigns**

2.  Select **New**

3.  For information on how to create recall campaigns, see [[mco-rc-my-campaigns|Create a campaign]].
