# Conversational Interfaces reference

Conversational Interfaces reference topics provide shared technical descriptions of features that apply to [[virtual-agent-landing-page|Virtual Agent]] and [[ci-use-agent-chat|Agent Chat]], such as the Interactions table.

-   **[[va-data-mgmt|Data management in Conversational Interfaces]]**  
Use data [[management|management]] tools, such as table cleaners, to maintain [[convint-landing-page|Conversational Interfaces]] tables.
-   **[[ci-interaction-insights-table|Interaction Insights table]]**  
User sentiment analysis prepares agents for the tone of incoming conversations. Business rules can be created based on this value and agents can take appropriate actions on the chat, whether it comes from Virtual Agent, another live agent, or from a queue.
