# Learning Core Reference

[[reference-hr-service-delivery-advanced-integration-with-workday|Reference]] topics provide additional information about [[learning-core_overview|Learning Core]].

-   **[Components installed with Learning Core](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/learning-core/learning-core-components-installed.md)**  
Several types of components are installed with activation of the Learning Core \[sn\_lc\] plugin, including user roles, and tables.
-   **[Course catalog form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/learning-core/course-catalog-table.md)**  
Use the [[course-catalog-table|course catalog form]] to create a new [[exploring-learning-exp|learning]] course catalog.
-   **[Learning library form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/learning-core/learning-library-form.md)**  
Use the [[learning-library-form|learning library form]] to [[create-learning-library|create a learning library]].
-   **[Learning task form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/learning-core/learning-task-form.md)**  
Assign a learning task to an employee that can be completed from the To-dos page in [[employee-center-landing-page|Employee Center]].
-   **[Life-cycle stages of a content collection in Learning Core](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/learning-core/lifecycle-stages-collection.md)**  
The State transition of a collection guides you through the stages of its life cycle.
-   **[Learning internal content form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/learning-core/learning-internal-content-form.md)**  
Use the [[learning-internal-content-form-exp|Learning internal content form]] to create internal Learning courses so that employees can learn new content and enhance their skills set.
-   **[Learning External Contents form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/learning-core/learning-external-contents-form.md)**  
Description of the field values in the [[learning-external-content-form|Learning External Content form]].
-   **[Collection form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/learning-core/collection-form-lc.md)**  
Use the [[collection-form-lc|Collection form]] to [[create-content-collection|create a content collection]] in Learning Core to dynamically group learning content that includes multiple internal and external courses.
