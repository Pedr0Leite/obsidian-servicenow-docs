# Enabling bookmarking screens and web pages

Configure the saved views feature to allow users to bookmark screens and web pages. Users can instantly access these saved views via an icon in the [[sg-mobile-tab-bar|navigation bar]].

The saved views feature is automatically enabled in the [[mobile-experience|Mobile Agent app]]. However, you must manually configure and enable it on the [[now-mobile-app|Now Mobile app]].

You can choose to disable access to this feature on individual screens.
