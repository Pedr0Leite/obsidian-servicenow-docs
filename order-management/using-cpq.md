# Using configure, price, quote applications

Learn how to use [[configure-price-quote-landing|configure, price, quote]] applications.

-   **[[using-servicenowcpq|Using the CPQ Configurator]]**  
Agents and customers use [[understand-the-commerce-logic-engine|the CPQ Configurator]] to customize [[configurable-products-explore|configurable products]] in [[order-mgt-overview|Sales Customer Relationship Management]] transactions such as opportunities, quotes, and orders.
-   **[[using-som-product-configurator|Using the legacy product configurator]]**  
The [[product-configurator|legacy product configurator]] in Sales Customer Relationship Management is an interface for customizing configurable product offers. The interface displays the product options available and automatically calculates product pricing as you select options.
-   **[[quote-mgmt-using|Using Quote Management]]**  
Use the [[quote-management|Quote Management]] application to build your quote. The interface includes the following tabs to help you create and manage product quotes.
-   **[[using-advanced-approval-management|Using Advanced Approval Management]]**  
Sales teams can use workflows for approving Sales Customer Relationship Management entities such as customer quotes. Workflows begin with approval requests submitted by sales agents, which are then routed for approval by designated approvers.
-   **[[using-post-sales-support|Using Customer Contracts and Entitlements]]**  
Learn how consumers, managers, agents, and administrators use the [[exploring-post-sales-support|Customer Contracts and Entitlements]] application to generate customer contracts and contract lines.
-   **[[sales-agreement-mgmt-using|Using Sales Agreement Management]]**  
Learn how managers, agents, and administrators use the [[sales-agreement-mgmt|Sales Agreement Management]] application to generate sales agreements from quotes.
-   **[[cpq-using|Using CPQ]]**  
Use ServiceNow CPQ to configure complex products, apply dynamic [[rules_101|rules]], and generate accurate quotes with speed and precision. You can understand how the CPQ experience works end-to-end—from guided configuration and real-time pricing to [[generating_documents_with_logik_io_configuration_data|document generation]] and seamless integration with the rules engine.
