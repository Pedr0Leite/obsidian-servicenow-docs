ServiceNow continues to innovate in the AI space with its new Now Assist Skill Kit, a powerful tool that allows developers and administrators to create custom generative AI skills tailored to their specific business needs. In a recent Creator Toolbox demo, ServiceNow product manager Eliza Orchard walked through the capabilities and setup process for this exciting new feature.

# **What is Now Assist Skill Kit?**

Now Assist Skill Kit, released in the Xanadu update, enables users to build custom generative AI skills that go beyond the out-of-box functionality provided by ServiceNow. While standard Now Assist features cover common use cases like incident resolution and knowledge article summarization, Skill Kit opens up possibilities for industry-specific and highly customized AI-powered workflows.

The key components of building a custom skill include:

1. Gathering and preparing input data

2. Creating a prompt to instruct the AI model

3. Leveraging a large language model (LLM) to generate the output

4. Deploying and activating the skill within your instance

# **A Real-World Example: Automated Knowledge Article Categorization**

To demonstrate Skill Kit's capabilities, Eliza walked through creating a custom skill to automatically categorize knowledge articles. This use case showcases how AI can streamline content management and improve findability of information.

The demo involved several key steps:

1. Using Flow Designer to retrieve existing knowledge base categories

2. Crafting a prompt in Skill Kit to instruct the AI on categorization

3. Configuring inputs to pass article content and category options to the model

4. Creating a UI action to trigger the skill from the knowledge article form

5. Testing and refining the skill's performance

# **Benefits and Use Cases**
