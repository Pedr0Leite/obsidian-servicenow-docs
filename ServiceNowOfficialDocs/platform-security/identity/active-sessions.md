# Viewing Active Sessions in Identity Center

Displays the information about the user sessions.

[[sc-active-sessions|Active sessions]] are the sessions that are open on the current  ServiceNow®  instance with different browsers or devices.

The Active Sessions tab in the [[identity-center|Identity Center]] helps you to [[identity-landing|identity]] your sessions based on the Browser, IP Address, and Sessions ID. Using this information you can take required actions such as extending or terminating the session.

Using this information can help determine if the sessions are real and not a security concern.
