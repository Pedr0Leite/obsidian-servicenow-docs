# Using Now Assist for Mobile

Use [[now-assist-mobile-landing|Now Assist for Mobile]] to enhance the mobile user experience.

-   **[[generate-work-order-res-notes|Generate work order resolution notes on mobile]]**  
Generate resolution notes in your Mobile app that summarize your work order task record.
-   **[[using-standard-chat-mobile|Using standard chat on mobile]]**  
Standard chat on mobile delivers an intuitive, text-based support experience within a static chat window, enhanced by AI-powered responses and integrated knowledge resources. You can interact with a Virtual Agent to resolve issues, access knowledge articles, and request catalog items all in one place.
-   **[[using-enhanced-chat-mobile|Using enhanced chat on mobile]]**  
Enhanced Chat on mobile offers a dynamic conversational support experience that integrates powerful AI-driven search and chat capabilities. You can seamlessly switch between multiple conversations, receive synthesized responses with inline citations, and access both knowledge articles and catalog items.
-   **[[using-premium-chat|Using premium chat]]**  
Premium Chat delivers a hybrid conversational experience that combines the flexibility of web-based Now Assist capabilities with the native mobile interface.
-   **[[using-na-web-search-mobile|Using Now Assist web search on mobile]]**  
Web search mode for Now Assist lets you perform an internet search from within Now Assist, showing external results from outside your organization.
-   **[[generate-a-record-summary-on-mobile|Generate a record summary on mobile]]**  
Generate a summary of a record in your [[now-mobile-app|Now mobile app]] using Now Assist.
-   **[[using-prominent-action-button|Using the prominent action button]]**  
Use the prominent action button to quickly access your Now Assist agent chat no matter where you're in the app.
-   **[[na-using-voice-to-text|Using Voice Input for mobile]]**  
The Voice Input feature allows users to easily transcribe spoken words into written text. Voice-to-text can be accessed directly from the search bar or Now Assist chat.
-   **[[using-people-genius-results|Using People Genius results on mobile]]**  
Search for people in your organization and view their profiles by entering their name into your search bar.
-   **[[using-mobile-ai-voice-agent|Using Mobile AI Voice Agent]]**  
Use [[mobile-voice-agent|Mobile AI Voice Agent]] to start voice conversations with an intelligent, AI-powered voice assistant.
