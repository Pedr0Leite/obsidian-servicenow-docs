# View Enrichment Results

View [[c_Observables|observables]], [[indicator|indicators]], and various objects enrichment results.

## Before you begin

Role required: sn\_sec\_tisc.admin

## Procedure

1.  Navigate to **Workspaces** &gt; **[[tisc-landing-page|Threat Intelligence Security Center]]** &gt; **Threat Intel Library** &gt; **Observables** &gt; **All Observables**.

2.  Select any observable\(s\) record.

3.  Go to **Enrichment Results** tab.

    The Enrichment Results displays the [[tisc-integrations|TISC integrations]] enrichment results.

    |Enrichment Name|Description|
    |---------------|-----------|
    |Threat Lookup Results|Lists all the associated [[tisc-threat-lookup|threat lookup]] results for an observable.|
    |Sightings|Lists all the associated [[indicator-sightings|sightings]] for an observable.|
    |Observable Enrichment Results|Lists all the associated [[tisc-observable-enrichment|observable enrichment]] results for an observable.|
