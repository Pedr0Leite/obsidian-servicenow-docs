# Delete a certification policy

Delete an unwanted certification policy permanently.

## Before you begin

Role required: sn\_cmdb\_admin + sn\_apm.apm\_analyst

## About this task

**Note:** Once a certification policy is deleted, the policy and all associated details are permanently removed and cannot be recovered.

## Procedure

1.  Navigate to **Workspaces** &gt; **[[ea-workspace|Enterprise Architecture Workspace]]**.

2.  Open the Data certification page by selecting the Data certification icon \(\[Omitted image "icon-data-cert.png"\] Alt text: data certification icon\).

    The Data certification page opens.

3.  Select the check box next to the policy that you want to delete and then select **Delete** from the context menu.

    A confirmation message appears.

    **Note:** This action permanently deletes the policy and it can't be undone.

4.  Select **Delete**.
