# System Diagnostics

Review System Diagnostics to monitor the health and performance of your instance.

Use the following tools and modules to diagnose the health of your instance:

-   [[events|System Events]] and Jobs Dashboard

    -   [[setup-event-monitoring-dashboard|System Events dashboard]]: Use the System Events dashboard to set the filters to display information about events on your instance.
    -   [[scheduled-jobs-dashboard|Scheduled Jobs dashboard]]: Use the [[c_ScheduledJobs|Scheduled Jobs]] dashboard to view the details of the completed jobs at the instance level.
    \[Omitted image "dashboard\_nav.gif"\] Alt text: Image showing dashboard gif

-   [[c_StatsToolsDiagnosticsPlugin|Stats Tools]]: Use Stats tools for performance evaluation by accessing the system activities statistics.
