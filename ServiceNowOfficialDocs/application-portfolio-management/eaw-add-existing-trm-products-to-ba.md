# Add existing Technology Reference Model products to business applications

You can add existing Technology Reference Model \(TRM\) product to business applications in the [[ea-workspace|Enterprise Architecture Workspace]].

## Before you begin

Role required: sn\_apm.apm\_analyst

## Procedure

1.  Navigate to **Workspaces** &gt; **Enterprise Architecture Workspace**.

2.  Open the Portfolio List view by selecting the Portfolio icon \(\[Omitted image "portfolio-icon.png"\] Alt text: Portfolio icon\).

3.  Select the expand row icon \(\[Omitted image "ExpandIcon.png"\] Alt text: Expand Row icon\) next to **Application Portfolio**.

4.  Select **Business Applications**.

5.  Select the business application that you want to add the existing TRM product to.

6.  Select **TRM Products**.

7.  Select **Add**.

8.  In the **Add TRM Product** pop-up window, select the existing TRM product.

9.  Select **OK**.


## Result

The existing TRM product is added to the business application.
