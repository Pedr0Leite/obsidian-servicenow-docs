# Manage Learning

Using [[exploring-learning-exp|Learning]], learning administrators can manage user activities. To aid the users learning journey, learning admins can create [[achievements-lxp|achievements]], define achievement rules, and monitor the overall learning [[activity-lxp|activity]] of a user.

-   **[Create a content collection](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/learning-core/create-content-collection.md)**  
[[create-content-collection|Create a content collection]] in Learning to dynamically group Learning content that includes multiple internal and external courses.
-   **[Create internal Learning content](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/learning-core/create-internal-learning-content.md)**  
Create internal Learning courses so that employees can learn new content and enhance their skills set.
-   **[View external Learning content](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/learning-core/view-external-learning-content.md)**  
View the details of all course items that are pulled from external sources into the Learning application.
-   **[Create a learning library](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/learning-core/create-learning-library.md)**  
[[create-learning-library|Create a learning library]] that includes learning content from internal and external content repositories.
-   **[Create a learning task](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/learning-core/create-learning-task.md)**  
Assign a learning task to an employee that can be completed from the To-dos page in [[employee-center-landing-page|Employee Center]].
-   **[Create an achievement definition](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/learning-core/create-achievement.md)**  
Create definitions for achievements to define the end goal.
-   **[Create user achievements](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/learning-core/create-user-achievements.md)**  
[[create-user-achievements|Create user achievements]] to project the accomplishments of course takers on [[overview-lxp|My Learning]].
-   **[Create an achievement rule](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/learning-core/create-achievement-rule.md)**  
Create achievement rules to define the criteria to be met to complete an achievement.
-   **[Create an achievement rule item](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/learning-core/achievement-rule-item.md)**  
[[achievement-rule-item|Create an achievement rule item]] to define if all or a count of items need to be completed to receive an achievement.
