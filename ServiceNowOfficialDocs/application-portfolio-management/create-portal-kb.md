# Create a portal knowledge base form

Create a new portal knowledge base to [[associate-portal-to-trm-cat|associate a portal with a knowledge base]] so that users can access the published TRM catalog from the selected portal.

|Field|Description|
|-----|-----------|
|Active|Select the check box to enable the mapping, or clear it to disable the portal and knowledge base mapping.|
|Knowledge Base|Select a knowledge base from the list.|
|Application|Uses the default value \(for example, Global\).|
|Order|Specify the display order for the knowledge base in the portal.|
|Portal|Select an existing portal to associate with the knowledge base.|
