# Create business capability form

Business Capability is a common table used within the [[application-portfolio-management-landing-page|Enterprise Architecture]] application.

## Create business capability form fields

<table id="table_cpz_5cm_2yb"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Name

</td><td>

Name of the business capability.

</td></tr><tr><td>

Parent

</td><td>

Name of the parent capability for the capability that you’re creating.

</td></tr><tr><td>

Level

</td><td>

Level at which the capability is in the hierarchy. This field is auto-populated and can't be edited.

</td></tr><tr><td>

Business Unit

</td><td>

Name of the business unit in the organizational structure.

</td></tr><tr><td>

Order

</td><td>

Position of the capability in the sequential order of all other business capabilities in that capability hierarchy.

</td></tr><tr><td>

Department

</td><td>

Name of the department in the organizational structure.

</td></tr><tr><td>

Owned by

</td><td>

User who owns the business capability.

</td></tr><tr><td>

Leaf Node

</td><td>

Option to select if the capability doesn’t have a child capability.

</td></tr><tr><td>

Hierarchy ID

</td><td>

Hierarchy ID of the capability. This field is auto-populated based on the order.

</td></tr><tr><td>

Description

</td><td>

A short description of the business capability.You can also elaborate or shorten the content in this field using Now Assist. For details, see [[elaborate-or-shorten-content-form-fields|Elaborate or shorten content in form fields]].

</td></tr></tbody>
