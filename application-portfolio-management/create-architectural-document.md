# Create an architectural document

Create an architectural document in the [[ea-workspace|Enterprise Architecture Workspace]]. An Architectural document captures and governs architecture information generated from diagrams.

## Before you begin

Role required: sn\_apm.apm\_user

## Procedure

1.  Navigate to **Workspace** &gt; **Enterprise Architecture Workspace**.

2.  Open the Portfolio List view by selecting the Portfolio icon \[Omitted image "portfolio-icon.png"\] Alt text: Portfolio icon.

3.  Select the expand row icon \(\[Omitted image "ExpandIcon.png"\] Alt text: Expand Row icon\) next to **Information Portfolio**.

4.  Select **Architectural Documents**.

5.  Select **New**.

6.  Enter a name and select an architectural category.

    **Note:** By default, the format is set to **Document**.

7.  Select **Create**.

    An artifact gets created.

    You can see the following tabs:

    -   Artifact content
    -   Details
    -   Related entities
    \[Omitted image "architectural-document.png"\] Alt text: Architectural document
