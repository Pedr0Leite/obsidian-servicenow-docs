# View value stream stages associated with a business capability

You can view the value stream stages associated with a business capability to understand which steps in your value stream the business capability supports.

## Before you begin

Role required: sn\_apm.apm\_user

## Procedure

1.  Navigate to **Workspace** &gt; **[[ea-workspace|Enterprise Architecture Workspace]]**.

2.  Open the Portfolio List view by selecting the Portfolio icon \[Omitted image "portfolio-icon.png"\] Alt text: Portfolio icon.

3.  Select the expand row icon \(\[Omitted image "ExpandIcon.png"\] Alt text: Expand Row icon\) next to **Business Architecture**.

4.  Select **Business Capabilities**.

5.  Select the business capability that you want to view the associated value stream stages for.

    The business capability details page appears.

6.  Select the **Value Stream Stages** tab.

    The list of value stream stages associated with the business capability is displayed.
