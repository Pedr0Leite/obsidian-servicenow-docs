# Export document templates

Download document templates in the required file format.

## Before you begin

Role required: sn\_apm.apm\_user

## About this task

## Procedure

1.  Navigate to **Workspaces** &gt; **[[ea-workspace|Enterprise Architecture Workspace]]**.

2.  Open the Setup page by selecting the Setup icon \(\[Omitted image "eaw-icon-setup.png"\] Alt text: setup icon.\).

3.  Select the expand row icon \(\[Omitted image "ExpandIcon.png"\] Alt text: expand row icon.\) next to **Templates**.

4.  Select **Document Templates**.

5.  Select **Export**.

6.  Select the required file format from the **File Type** drop-down list.

7.  Select the option to download or email from the **Delivery Type** drop-down list.

    If you select the delivery type as email, you must enter the email address.

8.  Select **Export**.

    The template data is downloaded in the selected file format.
