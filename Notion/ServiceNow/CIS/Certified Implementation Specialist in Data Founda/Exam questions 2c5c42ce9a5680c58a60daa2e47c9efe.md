# Exam questions

### **Sample Question – #1**

The Configuration Management team wants to identify which CIs have been created from an asset but have not been discovered. The team creates a report on the target CI Class.

How does the team identify these CIs in the report?

A.   All CIs with a Discovery source value of 'SNAssetManagement'

B.   All CIs with the Updated field older than a month

C.   All CIs where the Last Discovered field is empty

D.   All CIs with a Discovery source value of 'ServiceNow'

Answer: A

### **Sample Question – #2**

A State field is updated on an Asset record.   Which fields are automatically synchronized on the associated CI record?  *(choose two)*

A.  Operational status

B.   Install Status

C.  Hardware Status

D.  Asset tag

Answer: B, C

### **Sample Question – #3**

In a discussion with a CIO, a CMDB admin mentions that ServiceNow provides Playbooks which are structured guidelines included in different Data Foundations Dashboards. The admin needs to explain the outline of a Playbook.

Put the structure of Playbooks in sequential order.
