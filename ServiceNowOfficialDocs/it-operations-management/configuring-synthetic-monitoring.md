# Configuring synthetic monitoring

Plan and configure your implementation of [[synthetic-monitoring-landing-page|synthetic monitoring]].

## Configuration overview

-   **[[install-synthetic-monitoring|Install synthetic monitoring]]**  
You can install the synthetic monitoring application \(`com.snc.uib.sow_synthetics`\) with the admin role.
-   **[[create-synthetic-monitoring-locations|Create synthetic monitoring locations]]**  
Create a synthetic monitoring location to run a synthetic monitor. If you plan to run monitors from a ServiceNow hosted location, you can skip this procedure.
-   **[[create-synthetic-monitor|Create and edit a synthetic monitor]]**  
Create or edit a synthetic monitor to test the availability and performance of your HTTP endpoints before your users discover issues.
-   **[[create-synthetic-monitors-from-incidents|Create synthetic monitors from incidents]]**  
Create synthetic monitors from incidents to proactively track endpoint failures and reduce repeat occurrences.
-   **[[manage-synthetic-monitors|Manage synthetic monitors]]**  
Edit or delete synthetic monitors after creating them.
