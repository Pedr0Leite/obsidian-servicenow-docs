# Example widgets

All widgets in [[c_ServicePortal|Service Portal]] can be used as example code for how scripts are used in a widget. However, several base system widgets have been included expressly for this purpose.

Use the example widgets to see how to use HTML, CSS, or client and server scripts in Service Portal. You can also clone and extend each widget to suit your needs.

-   **[Announcements widget](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/service-portal/announcements-widget.md)**  
Users can view all active announcements. You can use this base system widget as-is in your portal or clone it to suit your own business needs.
-   **[Approvals widget](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/service-portal/approvals-widget.md)**  
Users can approve or reject items directly within Service Portal. You can use this base system widget as-is in your portal or clone it to suit your own business needs.
-   **[Approval Info Record widget](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/service-portal/approval-info-record-widget.md)**  
The [[approval-info-record-widget|Approval Info Record widget]] shows details about the approval request and a full record for an approval including the [[activity-stream-configurable-workspace|activity stream]].
-   **[Approval Info widget](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/service-portal/approval-info-widget.md)**  
The [[approval-info-widget|Approval Info widget]] works in tandem with the Approval widget to display details about the approval request. You can use this base system widget as-is in your portal or clone it to suit your own business needs.
-   **[Approval Record widget](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/service-portal/approval-record-widget.md)**  
The [[approval-record-widget|Approval Record widget]] shows the full record for an approval including the activity stream. You can use this base system widget as-is in your portal or clone it to suit your own business needs.
-   **[Breadcrumbs widget](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/service-portal/breadcrumbs-widget.md)**  
The [[breadcrumbs-widget|breadcrumbs widget]] allows users to easily navigate around a portal. You can use this base system widget as-is in your portal or clone it to suit your own business needs.
-   **[Breakout Game widget](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/service-portal/breakout-game-widget.md)**  
Add a fun, interactive 404 page to pages that do not exist using the [[breakout-game-widget|Breakout Game widget]]. You can use this base system widget as-is in your portal or clone it to suit your own business needs.
-   **[Calculator widget](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/service-portal/calculator-widget.md)**  
The [[calculator-widget|calculator widget]] does simple calculations. You can use this base system widget as-is in your portal or clone it to suit your own business needs.
-   **[Carousel widget](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/service-portal/carousel-widget.md)**  
Showcase specific items in your catalog using a scrolling list of images in the [[carousel-widget|carousel widget]]. You can use this base system widget as-is in your portal or clone it to suit your own business needs.
-   **[Change Password widget](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/service-portal/change-password-widget.md)**  
Users can change their passwords using the [[change-password-widget|Change Password widget]].
-   **[Cool Clock widget](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/service-portal/cool-clock-widget.md)**  
Show different times around the world using the [[cool-clock-widget|Cool Clock widget]]. You can use this base system widget as-is in your portal or clone it to suit your own business needs.
-   **[Data table from instance definition widget](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/service-portal/data-table-widget.md)**  
Display a filtered list on your portal using the [[data-table-widget|data table from instance definition widget]]. From the widget's context menu, portal users can export the filtered list. You can use this base system widget as-is in your portal or clone it to suit your own business needs.
-   **[Data Table from URL definition widget](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/service-portal/data-table-url-widget.md)**  
The [[data-table-url-widget|Data Table from URL definition widget]] displays the table you select from the list. You can use this base system widget as-is in your portal or clone it to suit your own business needs.
-   **[Documents widget](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/service-portal/documents-widget.md)**  
You can add, edit, or delete documents associated with records based on user permissions through the [[documents-widget|Documents widget]]. You can use this widget in your portal or clone and customize it.
-   **[Form widget](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/service-portal/form-widget.md)**  
The [[form-widget|form widget]] is a platform form within the Service Portal UI with a few differences. You can use this base system widget as-is in your portal or clone it to suit your own business needs.
-   **[Header menu widget](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/service-portal/header-widget.md)**  
The [[header-widget|Header Menu widget]] controls which options appear in the page header. You can use this base system widget as-is in your portal.
-   **[Hello World widgets](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/service-portal/hello-world-widget.md)**  
The [[hello-world-widget|Hello World widgets]] are included with Service Portal as examples of how to use and create widgets. You can use this base system widget as-is in your portal or clone it to suit your own business needs.
-   **[HTML widget](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/service-portal/html-widget.md)**  
Use the [[html-widget|HTML widget]] to directly inject HTML, text, [[lists-configurable-workspace|lists]], or content in general into a page. You can use this base system widget as-is in your portal or clone it to suit your own business needs.
-   **[Icon Link widget](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/service-portal/icon-link-widget.md)**  
Link to any other item. You can use this base system widget as-is in your portal or clone it to suit your own business needs.
-   **[Icon menu list widget](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/service-portal/icon-menu-list-widget.md)**  
A simple list with a glyph icon next to each link. You can use this base system widget as-is in your portal or clone it to suit your own business needs.
-   **[Language Switch widget](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/service-portal/language-switch-widget.md)**  
Add the [[language-switch-widget|Language Switch widget]] to a landing or homepage to allow your users to change the language of the page. You can use this base system widget as-is in your portal or clone it to suit your own business needs.
-   **[Link button widget](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/service-portal/link-button-widget.md)**  
The [[link-button-widget|Link Button widget]] is a button you can nest in any other widget that links to another destination. You can use this base system widget as-is in your portal or clone it to suit your own business needs.
-   **[Login widget](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/service-portal/login-widget.md)**  
The [[login-widget|login widget]] controls user access to your site. You can use this base system widget as-is in your portal or clone it to suit your own business needs.
-   **[My Requests widget](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/service-portal/my-requests-widget.md)**  
The [[my-request-widget-portal|My Requests widget]] stores all of your open requests in one place. You can use this base system widget as-is in your portal or clone it to suit your own business needs.
-   **[My Requests widget](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/service-portal/my-request-widget-portal.md)**  
Use the My Requests widget \(my-requests-v2\) to enable requesters to view open or closed requests in Service Portal.
-   **[Organization Chart widget](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/service-portal/org-chart-widget.md)**  
The [[org-chart-widget|Organization Chart widget]] shows employees in a tree structure relative to their manager. You can use this base system widget as-is in your portal or clone it to suit your own business needs.
-   **[New Organization Chart widget](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/service-portal/new-organization-chart-widget.md)**  
The [[new-organization-chart-widget|New Organization Chart widget]] shows employees in a tree structure relative to their manager. You can use this widget in your portal or duplicate it to suit your own business needs.
-   **[Simple List widget](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/service-portal/simple-list-widget.md)**  
The [[simple-list-widget|Simple List widget]] can be used to display any list in the system within Service Portal. You can use this base system widget as-is in your portal or clone it to suit your own business needs.
-   **[Sample Footer widget](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/service-portal/sample-footer-widget.md)**  
The [[sample-footer-widget|Sample Footer widget]] is an example of a footer you can use in your portal. You can use this base system widget as-is in your portal or clone it to suit your own business needs.
-   **[Ticket Attachments widget](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/service-portal/ticket-attachment-widget.md)**  
Use the attachment widget to attach items to tickets. You can use this base system widget as-is in your portal or clone it to suit your own business needs.
-   **[Ticket Conversations widget](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/service-portal/ticket-conversations-widget.md)**  
Record of ticket items. Users can use this widget to communicate back and forth with the fulfiller and the receiver. You can use this base system widget as-is in your portal or clone it to suit your own business needs.
-   **[Ticket Fields widget](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/service-portal/ticket-fields-widget.md)**  
The [[ticket-fields-widget|Ticket Fields widget]] displays information about a request that a user has made. You can use this base system widget as-is in your portal or clone it to suit your own business needs.
-   **[Ticket Location widget](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/service-portal/ticket-location-widget.md)**  
Share your location in a ticket. You can use this base system widget as-is in your portal or clone it to suit your own business needs.
-   **[User Profile widget](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/service-portal/user-profile-widget.md)**  
Display user profile information. You can use this base system widget as-is in your portal or clone it to suit your own business needs.
