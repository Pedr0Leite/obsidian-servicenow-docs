# Archiving records in Core UI

Manage table size growth and improve query performance by archiving records.

You can archive records in core tables such as the Task \[task\] table and records in [[custom-tables|custom tables]] that you create on the ServiceNow AI Platform using archive rules.

## Overview of archive rules in Core UI

Create and manage archive rules in Core UI.

-   [[t_CreateAnArchiveRule|Create an archive rule in Core UI]]

    Define the criteria for archiving records by creating an archive rule.

-   [[t_ArchiveRelatedRecords|Archive related records in Core UI]]

    Archive, clear, or delete related records from an archive rule.

-   [[managing-archived-data|Managing archived data in Core UI]]

    Change the schedule for an archive rule, stop the archive rule from running, or restore your archived data.
