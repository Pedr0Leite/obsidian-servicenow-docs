# Assigning a territory on account

You can update the primary territory on an account in which Sales Territory Management​ has been implemented.

## Before you begin

Role required: sales territory admin \(or a user with sales territory override role\)

## Procedure

1.  In the CSM Configurable Workspace, select the **List** \[Omitted image "list-outline-24.svg"\] Alt text: view.

2.  Navigate to **Customer** &gt; **Accounts** and select the account you want to update the territory.

3.  On the Details tab, update the **Sales Ownership** &gt; **Primary Sales Territory** field with the other available territories.

4.  Select **Save**.
