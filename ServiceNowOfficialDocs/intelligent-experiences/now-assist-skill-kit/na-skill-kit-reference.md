# Now Assist Skill Kit reference

Reference topics for [[now-assist-skill-kit-landing|Now Assist Skill Kit]]

-   **[Now Assist Skill Kit roles](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/intelligent-experiences/now-assist-skill-kit/na-skill-kit-roles.md)**  
Certain roles are required to use Now Assist Skill Kit functionality.
-   **[Field of use for Now Assist Skill Kit](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/intelligent-experiences/now-assist-skill-kit/nask-field-of-use.md)**  
Now Assist Skill Kit is included in various [[platform-now-assist-landing|Now Assist]] packages that cover a given customers’ ability to build [[now-assist-skills|Now Assist skills]].
-   **[Web search custom skill](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/intelligent-experiences/now-assist-skill-kit/web-search-custom-skill.md)**  
The [[web-search-custom-skill|web search custom skill]] performs an internet search to answer a query. Web search is used whenever the LLM and AI Search are unable to provide results or whenever web search mode is activated.
