# Associate an Architectural Artifact to a TRM product

As an Enterprise Architect, you can associate an architectural artifact to a TRM product.

## Before you begin

You must be part of the Enterprise Architect Group.

Role required: sn\_apm.apm\_analyst

## Procedure

1.  Navigate to **Workspace** &gt; **[[ea-workspace|Enterprise Architecture Workspace]]**.

2.  Open the Technology Portfolio page by selecting the Technology Portfolio icon \[Omitted image "technology-portfolio-icon.png"\] Alt text: Technology portfolio icon.

3.  Open any existing TRM product.

4.  Select **New**.

5.  On the form, fill in the fields.

    For field information, see [[eaw-new-arch-artifact-form|Create architectural artifact form]].

    To edit any existing artifact, select the artifact and make the necessary changes.

6.  Select **Submit**.

    An architectural artifact gets created and associated to the TRM product. The Related Entities tab of the artifact shows the TRM product associated to it.
