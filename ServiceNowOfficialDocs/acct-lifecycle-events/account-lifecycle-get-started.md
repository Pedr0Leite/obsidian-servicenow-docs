# Getting started with Customer Success Management

Learn how to activate [[account-lifecycle-events-landing|Customer Success Management]] and review the plugins that are activated with it.

-   **[[account-lifecycle-activate|Activate Customer Success Management]]**  
The [[account-lifecycle-events-customer-success-about|Customer Success]] Management \(com.sn\_acct\_lc\) plugin is available as a separate subscription. Activating this plugin also activates the related plugins required to use the application.
-   **[[account-lifecycle-required-plugins|Plugins activated with Customer Success Management]]**  
Activating [[account-lifecycle-use-cust-success|Customer Success]] Management installs the following related plugins, if they aren't already active.
-   **[[account-lifecycle-read-only|Read-only field enhancements]]**  
Read-only field protections have been moved from the client side to the server side. This prevents users from manipulating data through client-side methods.
