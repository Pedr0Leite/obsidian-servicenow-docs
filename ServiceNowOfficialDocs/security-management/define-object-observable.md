# Define object-observable relationships

Define relationships between SDOs and the observable object \(SCO\).

## Before you begin

Role required: sn\_ti.admin

## Procedure

1.  Navigate to **All** &gt; **[[threat-intel-landing-page|Threat Intelligence]]** &gt; **[[ioc-repository|IoC Repository]]** &gt; **Object-Observable-Relationships**.

2.  Click **New**.

3.  Complete the fields in the form as appropriate.

    |Field|Description|
    |-----|-----------|
    |**Object**|Select and define the source object.|
    |**Type**|Enter a descriptive name to identify the object type.|
    |**Observable**|Select and define the observable.|
    |**Relationship Type**|A description that provides more details and context about the relationship type.|
    |**Source**|Specifies the threat source from which this record is created.|
    |**Source ID**|Uniquely identifies the source object.|

4.  Click **Submit**.
