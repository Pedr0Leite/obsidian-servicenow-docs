# Create an issue for model

Add or create model issues and their remediation actions. Capture validation findings or performance problems and follow them through to resolution.

## Before you begin

Role required: sn\_model\_risk\_mgmt.model\_risk\_coordinator, sn\_model\_risk\_mgmt.model\_governance

## Procedure

1.  Navigate to **All** &gt; **[[model-risk-workspace|Model Risk Workspace]]**.

2.  Select the list icon \[Omitted image "list-icon-mrm.png"\] Alt text:.

3.  From the list, navigate to **Models** &gt; **All**.

4.  Select the model and navigate to the **Issues** tab.

5.  Select **New**.

6.  On the form, fill in the fields.

    For a description of the field values on the Create an issue form, see [[create-an-issue-form-mrm|Create an issue form]].

7.  Select **Save**.
