# MCO Integration APIs

Use ServiceNow APIs to change functionality and add features in the [[manufacturing-overview|Manufacturing Commercial Operations]] \(MCO\) solution.

-   **[[warranty-claims-SOAP-API|Warranty Claims SOAP API]]**  
The Warranty Claims SOAP API handles incoming repair orders from dealer systems and converts them into warranty claims within the Oracle Enterprise Manager \(OEM\)’s platform.
