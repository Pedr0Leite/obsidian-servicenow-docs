# Agentic development on the ServiceNow AI Platform

Use ServiceNow AI Platform environments, products, tools, and skills to develop, expand, test, and deploy apps using a conversational interface. Autonomous app development on the ServiceNow AI Platform is agentic development.

\[Omitted image "vc-approaches.png"\] Alt text: Agentic development approaches from autonomous to developer-controlled

-   **[[vibe-coding-environments|Agentic development in ServiceNow AI Platform environments]]**  
You can develop apps agentically with AI tools on the ServiceNow AI Platform in a safe, isolated environment using a non-production instance with [[sandboxes-landing|Developer Sandboxes]], or on a Personal Developer Instance \(PDI\).
-   **[[vc-build-agent-landing|AI-assisted ServiceNow AI Platform development with Build Agent]]**  
[[build-agent|Build Agent]] is the primary AI-powered development tool that works directly with the ServiceNow AI Platform to help you build applications. It generates code based on natural language descriptions, suggests improvements, and automates repetitive development tasks while you retain control of your application.
-   **[[vibe-code-now-assist-creator|AI-assisted app creation with Now Assist for Creator]]**  
[[now-assist-for-creator-landing|Now Assist for Creator]] provides core skills for rapid conversational app development. Use the integrated tools to generate scripts, optimize code, scaffold applications, and automate testing through natural language prompts.
-   **[[vibe-code-now-assist-app-engine|AI-assisted development with Now Assist for App Engine]]**  
[[add-ai-to-custom-apps-with-now-assist-for-app-engine-enterprise|Now Assist for App Engine]] enables you to embed AI functionality into an application that you built on the ServiceNow AI Platform.
-   **[[vc-and-atf|Testing agentically developed apps]]**  
As part of the agentic development process on the ServiceNow AI Platform, one of the key features is the automatic generation of [[atf-landing-page|Automated Test Framework \(ATF\)]] [[tests-module|tests]].
-   **[[vc-and-deployment|Agentic development and deployment]]**  
After you generate and expand an app on the ServiceNow AI Platform, the [[get-started-deployment|deployment]] process is tightly integrated into the workflow.
