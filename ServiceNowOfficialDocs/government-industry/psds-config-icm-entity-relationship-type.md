# Configure relationship types for an entity record in Investigative Case Management

As an admin, you can add or remove relationship types used to classify the relationship between two entities in a case.

## About this task

For more information about relating entities to a case, see [[psds-using-icm-add-entities|Add entities to an investigative case]].

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Task Relationships** &gt; **Relationship Types**.

2.  Select **New**.

3.  On the form, fill in the fields.

4.  Select **Submit**.


## Result

The relationship type definition is now created, and will display in the relationship type dropdown menu when an investigator moves to associate an entity to a case, or an entity to an entity.

## What to do next

An investigator can use this relationship type to define the relationship between an entity record and a case, or between two entities. For more information on how to link entities to a case, see [[psds-using-icm-associate-evidence-entities|Associate and link evidence to entities]]. For more information on how to link entities to each other, see [[psds-using-icm-associate-entites|Associate and link multiple entities]].
