# Administration in Unified Security Exposure Management

The Administration for Security Exposure Management application offers a unified administrative experience across all [[unified-security-exposure-management-landing-page|Unified Security Exposure Management \(USEM\)]] applications. It introduces the Admin Console within the Security Exposure Management \(SEM\) Workspace, allowing administrators to configure Security Exposure Management applications and monitor integrations from a single [[location|location]].

The Administration Console centralizes configuration and administration for all USEM applications, streamlining the management process and ensuring consistent settings across the environment. The following modules are available within the Admin Console:

-   **Integrations**: The Integration status page provides a comprehensive overview of installed third-party applications and their integration runs. Real-time status updates enable administrators to diagnose and resolve issues related to failed integration runs. For more information, see [[review-usem-integrations|Review Unified Security Exposure Management integrations]].
-   **Configurations**: Configuring rules automates, organizes, and manages the lifecycle of findings, ensuring scalability, data consistency, and faster response times by reducing manual intervention. For more information, see [[sem-configure-rules-manage-findings|Configure rules to manage findings]].
-   **Additional Configuration Options**
    -   **[[tisc-email-notifications|Email Notifications]]**: Set up email notifications to share important updates and activities, such as approval and rejection of false-positive requests. Specify when to send notifications, who receives them, and their content. For more information, see [[sem-configure-email-notifications|Configure email notifications in Unified Security Exposure Management]].
    -   **Email Templates**: Create and manage preconfigured email templates for sending notifications. For more information, see [[sem-configure-email-templates|Configure email templates in Unified Security Exposure Management]].
    -   **Severity [[mapping-logrhythm|Mapping]]**: The [[sem-workspace-user-interface|Security Exposure Management Workspace]] severity mapping transforms third-party source severity fields to recognizable fields in Unified Security Exposure Management. For more information, see .
    -   **[[sem-visualization-library|Visualization Library]]**: Configure the Visualization library to select columns for the Findings view page, including available widgets, activation status, and additional details. For more information, see [[sem-configure-visualization-library|Configure Visualization library]].
