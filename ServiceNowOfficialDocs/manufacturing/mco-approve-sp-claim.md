# Approve all claims

As an OEM claims agent, approve all the claim request raised by the dealer.

## Before you begin

Role required: manufacturing operations admin \(sn\_mfg\_cmn.manufacturing\_operations\_admin\) or claims agent \(sn\_claim\_cmn.claims\_agent\)

## Procedure

1.  Navigate to **Workspaces** &gt; **CSM/FSM Configurable Workspace** &gt; **Lists** &gt; **[[mco-sales-promotion|Sales promotion]]**.

    **Note:** To create and navigate to the Review &amp; approve form, see [[mco-create-sls-pro-claim-case|Create a sales promotion claim case]].

2.  Select **Review &amp; approve** from **Activities**.

3.  Select **\[Omitted image "check-outline-24.svg"\] Alt text: Check**

4.  Add **Notes**.

    All the claims are approved and **Claim Summary** activity is displayed.

5.  Select **Complete**.

6.  View the sales promotion claim summary details by selecting **Claim Summary**
