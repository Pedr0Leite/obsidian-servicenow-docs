# Configure Zoom and touchpoint integration

Integrate Zoom with touchpoint records to schedule virtual meetings and generate AI-powered meeting summaries.

This integration uses the Zoom spoke and a bi-directional webhook to connect Zoom meetings to touchpoint records. After the integration is set up, [[account-lifecycle-events-customer-success-about|customer success]] managers can schedule Zoom meetings from a touchpoint and automatically generate summaries when meetings end.

-   **[[account-lifecycle-config-zoom-sum|Configure Zoom to create a touchpoint meeting summary]]**  
Set up the Zoom spoke integration and the bi-directional web hook to generate summaries for virtual touchpoint meetings.
