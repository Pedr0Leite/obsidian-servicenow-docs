# Introducing Custom Record Summarization: Now Assist for App Engine's Inaugural Skill

ServiceNow has launched an exciting new capability with Now Assist for App Engine (version 28.2.4+), and it's called Custom Record Summarization. If you've ever found yourself scrolling through lengthy custom application records trying to get the gist of what's going on, this AI-powered feature is about to become your new best friend. In essence, it generates intelligent summaries of records in your custom tables and applications, helping you quickly understand the important details and context without the manual effort. Whether you're managing a PTO request system, tracking project milestones, or overseeing any custom workflow, Now Assist can now give you the highlights in seconds.

Flexible Configuration for Your Needs

What makes this skill particularly powerful is its flexibility. During configuration, you get to choose exactly which tables and fields Now Assist should summarize, and you can even pull in additional context from:

Related tables
Relationships
Emails
Attachments
The key to getting accurate summaries is providing good field descriptions during setup—think of these as instructions that help the AI understand what each field represents. For example, if you're summarizing a PTO request table, you'd describe fields like "PTO start date" with details such as "Date that the employee's PTO begins. Date is in MM-DD-YYYY format." The more context you provide, the better your summaries will be.

Choose How Users Access Summaries

From a user experience perspective, you have options for how people interact with these summaries:

In-product: Display summaries directly on forms and workspaces for quick reviews on record pages
Now Assist panel: Access summaries through the chat interface for conversational workflow
Both: Enable summaries in multiple locations for maximum flexibility
Access Control & Usage Considerations

Access control is built right into the configuration process, so you can define which users and groups can generate summaries based on roles and conditions. You can even customize access differently for each display option.

Important note: If you're including attachments in your summaries, this will consume more of your Now Assist usage, so it's worth monitoring through Subscription Management.

Overall, Custom Record Summarization represents a significant step forward in making custom applications more intuitive and efficient, and it's just the beginning of what Now Assist for App Engine can do.
