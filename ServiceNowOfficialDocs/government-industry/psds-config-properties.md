# Public Sector Digital Services properties

Use system properties to customize the Public Sector Digital Services application.

These properties control settings for several different features, including communication, contact relationships, and access to playbook and application data. You must have the admin role to configure these properties.

You can also access Public Sector Digital Services properties using the system property \[sys\_properties\] table. Start by entering **sys\_properties.list** in the application navigator and selecting **Enter**. You can then filter the properties by application.

For more information on the properties that are installed with Public Sector Digital Services, see [[properties-installed-with-psds|Properties installed with Public Sector Digital Services]]
