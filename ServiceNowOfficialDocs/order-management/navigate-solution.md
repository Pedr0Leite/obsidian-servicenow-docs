# Navigate within a solution

Use the solution navigation sidebar to move between configurations in a solution session.

## Before you begin

You are in an active solution session with at least one child configuration. For more information, see [[add-configurable-products-solution|Add configurable products to a solution]].

Role required: none

## Procedure

1.  In the solution navigation sidebar, review the list of [[configurable-products-explore|configurable products]] in the current session.

2.  Select the configurable product you want to navigate to.

    The selected configuration becomes the active view.


## Result

You can also navigate to a configuration by selecting its configurable product entry in the solution BOM.

## What to do next

After navigating to a configuration:

-   To review all products added across the session, see [[view-solution-bom|View the solution bill of materials]].
-   To remove a child configuration that is no longer needed, return to the parent configuration and change the triggering condition. See [[remove-child-configuration|Remove a configurable product from a solution]].
