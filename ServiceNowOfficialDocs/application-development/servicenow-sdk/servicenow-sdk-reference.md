# ServiceNow SDK reference

Reference topics provide additional information about [[servicenow-sdk-cli-commands|ServiceNow SDK CLI]] commands and [[servicenow-fluent|ServiceNow Fluent]] APIs.

-   **[ServiceNow SDK CLI](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/servicenow-sdk/servicenow-sdk-cli-commands.md)**  
Use the [[servicenow-sdk-landing|ServiceNow SDK]] command-line interface \(CLI\) to manage changes between a local application and the application on an instance.
-   **[ServiceNow Fluent API reference](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/servicenow-sdk/servicenow-fluent-api-reference.md)**  
Use ServiceNow Fluent APIs to define the metadata that makes up scoped applications in source code with the [[servicenow-ide-landing|ServiceNow IDE]] or ServiceNow SDK.
