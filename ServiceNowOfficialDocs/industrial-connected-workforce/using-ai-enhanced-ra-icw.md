# Using AI Enhanced recommended actions for Industrial Connected Workforce

After you [[configure-ai-enhanced-ra-icw|configure AI Enhanced recommended actions for Industrial Connected Workforce]], you can use the feature to contextualize external sources related to a deviation and generate an action plan.

-   **[[use-ai-enhanced-ra-icw|Contextualize an external document for a Deviation in the Digital Factory Workspace]]**  
With the AI Enhanced recommended actions for Industrial Connected Workforce, contextualize an external document to get an explanation of why the document is relevant to a deviation.
-   **[[generate-action-plan-ai-enhanced-ra-icw|Generate an action plan to resolve a Deviation in the Digital Factory Workspace]]**  
Generate an action plan to document and track remediation steps for an [[industrial-connected-workforce-overview|Industrial Connected Workforce]] \(ICW\) deviation. Action plans help you organize tasks, assign responsibilities, and monitor progress toward resolution.
