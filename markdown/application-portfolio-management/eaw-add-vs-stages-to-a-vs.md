# Add value stream stages to a value stream

You can add value stream stages to a value stream to break it down into discrete steps and establish the foundation for associating business processes and business capabilities with each stage.

## Before you begin

Role required: sn\_apm.apm\_user

## Procedure

1.  Navigate to **Workspace** &gt; **[[ea-workspace|Enterprise Architecture Workspace]]**.

2.  Open the Portfolio List view by selecting the Portfolio icon \[Omitted image "portfolio-icon.png"\] Alt text: Portfolio icon.

3.  Select the expand row icon \(\[Omitted image "ExpandIcon.png"\] Alt text: Expand Row icon\) next to **Business Architecture**.

4.  Select **Value Streams**.

5.  Select the value stream that you want to add value stream stages to.

    A new page appears, displaying the details of the value stream.

6.  Select the **Value Stream Stages** tab.

7.  Select **New**.

8.  On the form, fill in the field.

    For a description of the field values, see [[eaw-create-new-value-stream-stage-form|Create value stream stage form]].

9.  Select **Save**.
