# Delete model categories

If a model category is no longer needed, you can delete it.

## Before you begin

Role required: model\_manager

## About this task

Only [[model-hierarchy|model categories]] that are not referenced by any [[c_Models|models]] can be deleted.

## Procedure

1.  Navigate to **All** &gt; **[[c_ProductCatalog|Product Catalog]]** &gt; **Product Model** &gt; **Model Categories**.

2.  Select the check box beside the model category **Name**.

3.  From the **Actions on Selected Rows** menu below the list, click **Delete**.

4.  Click **Delete**.
