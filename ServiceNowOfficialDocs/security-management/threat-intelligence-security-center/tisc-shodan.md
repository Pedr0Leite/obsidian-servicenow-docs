# Shodan integration

Configure [[shodan-lookups|Shodan integration]] to enable automated discovery and analysis of internet-connected devices in your network [[threat-intelligence-infrastructure|infrastructure]].

The Shodan integration performs enrichment on the following [[c_Observables|observables]]:

-   IP addresses
-   URLs
-   Domains

-   **[Configure and enable Shodan integration](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/security-management/threat-intelligence-security-center/shodan-integration.md)**  
Before you use Shodan integration, you must download it from the ServiceNow Store.
