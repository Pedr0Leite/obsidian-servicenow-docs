# Create a cost type for Application TCO in Enterprise Architecture Workspace

Create a cost type as capital expense \(Capex\) or operating expense \(Opex\) for Application TCO. Specify the type of cost being ingested in [[ea-workspace|Enterprise Architecture Workspace]].

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **Workspaces** &gt; **Enterprise Architecture Workspace** &gt; **Setup**.

2.  Select **TCO**.

3.  Select **Cost types**.

4.  Select **New**.

5.  Enter a name and select the expense type \(Capex or Opex\).

6.  Select **Save**.
