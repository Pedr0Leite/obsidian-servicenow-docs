# View move tasks

Monitor the move-related tasks that are created in the application to fulfill a move request. View the move tasks assigned to you, the tasks that are currently open, and all the move tasks created in the application.

## Before you begin

Role required: sn\_wsd\_move.manager

## Procedure

1.  Navigate to **All** &gt; **[[workplace-move-mgmt-overview|Workplace Move Management]]** &gt; **Move tasks**.

2.  To view move tasks, select any of the following options:

    -   **Assigned to me**: Select this option to view the move tasks that are assigned to you.
    -   **Open tasks:** Select this option to view open move tasks.
    -   **All move tasks**: Select this option to view all the move tasks.
