# Add associated skills to an achievement

[[add-associated-skills|Add associated skills]] to define the scope of an achievement.

## Before you begin

Role required: sn\_lep.achievement\_manager

Create an achievement. For more information, see [Create an achievement definition](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/learning-core/create-achievement.md).

## Procedure

1.  Navigate to **All** &gt; **[[exploring-learning-exp|Learning]]** &gt; **[[achievements-lxp|Achievements]]** &gt; **Achievement definitions**.

2.  Select the achievement that you want to add associated skills to.

3.  On the Associated skills related list, select **Edit**.

4.  Add the required skills from the collection to the associated skills list.

5.  Select **Save** and **Update** the achievement definition.
