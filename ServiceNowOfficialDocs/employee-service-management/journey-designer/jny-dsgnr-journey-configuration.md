# Create and configure Journey designer journeys

[[jny-dsgnr-create-journey|Create a journey]] and embed the necessary components from [[hr-lifecycle-events-landing-page-1|Lifecycle Events]] and [[journey-accelerator-1|Journey Accelerator]].

## Before you begin

Role required: sn\_jny.admin

## About this task

A journey is a top-level container, where you define whether the Journey contains Lifecycle Events or Journey Accelerator modules, as well as the permissions for journey owners and mentors.

## Procedure

1.  Navigate to **All** &gt; **[[jny-dsgnr-landing-page-1|Journey designer]]** &gt; **Manage Journey Configurations**.

2.  Select **New**.

3.  On the form, fill in the fields.

    See [Journey designer Journey Configuration fields reference](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/journey-designer/jny-dsgnr-jrny-config-fields.md).

4.  Click **Submit**.
