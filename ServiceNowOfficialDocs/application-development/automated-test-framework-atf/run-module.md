# Run

Start a [[atf-client-test-runner-module|client test runner]] and view information about test runners and test runs.

For information on how to work with test runners, see [Working with client test runners](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/automated-test-framework-atf/atf-test-runners.md).

-   **[Client test runner](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/automated-test-framework-atf/atf-client-test-runner-module.md)**  
The Client Test Runner opens a browser window or tab for running manually-started client automated [[tests-module|tests]].
-   **[Scheduled client test runner](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/automated-test-framework-atf/atf-sched-test-runner-module.md)**  
The Scheduled Client Test Run opens a browser window for running scheduled client automated tests.
-   **[Active manual test runners](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/automated-test-framework-atf/atf-active-manual-runners-module.md)**  
View the client test runners table filtered to show only those runners available to run manually-started tests.
-   **[Active scheduled test runners](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/automated-test-framework-atf/atf-active-sched-runners-module.md)**  
View the client test runners table filtered to show only those runners available to run tests to be started by a schedule.
-   **[Waiting/running test runs](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/automated-test-framework-atf/waiting-running-test-runs.md)**  
The Waiting/Running Test Run module opens a list of records showing the tests waiting to be run.
-   **[Waiting/running suite runs](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/automated-test-framework-atf/waiting-running-suite-runs-module.md)**  
The [[waiting-running-suite-runs-module|Waiting/Running Suite Runs]] module opens a list of records showing the test [[suites-module|suites]] waiting to be run.
-   **[Allowed client error records](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/automated-test-framework-atf/whitelisted-client-error-records.md)**  
Review the list of existing Allowed Client Error \[sys\_atf\_whitelist\] records to see which client errors produce warnings and which are ignored. Modify existing [[whitelisted-client-error-records|Allowed Client Error records]] as needed or create new ones.
-   **[Reported client errors](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/automated-test-framework-atf/reported-client-errors.md)**  
The Reported Client Error module lists test logs across all tests that are client errors and have failed. You can review individual test log records, and [[whitelisted-client-errors|allowed client errors]] as needed to prevent them from adversely impacting future test executions.
