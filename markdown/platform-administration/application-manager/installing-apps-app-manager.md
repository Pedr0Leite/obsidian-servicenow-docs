# Installing applications, plugins, and products

[[application-manager|Application Manager]] enables you to install any applications, plugins, or products that are included in your license or that have been procured from the [[servicenow-store|ServiceNow Store]].

## Overview of using the Application Manager to install

You can view all applications, plugins, and products available for your instance by navigating to the Application Manager and selecting the **Available for you** tab. From there, you can install new applications, products, and plugins to your instance.

**Note:** Some applications and plugins can't be uninstalled after they've been installed.

-   **[Installation details](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-administration/application-manager/installation-details.md)**  
When you install or update applications or products, the Application Manager displays [[installation-details|installation details]] for you to review. Use installation details to verify version and dependency details.
-   **[Entitle an application](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-administration/application-manager/entitle-applications-in-application-manager.md)**  
[[entitle-applications-in-application-manager|Entitle an application]] from your ServiceNow AI Platform instance if the application has been purchased but not yet linked to the instance.
-   **[Install an application or plugin](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-administration/application-manager/installing-applications-in-application-manager.md)**  
[[installing-applications-in-application-manager|Install an application or plugin]] to make its features available on your instance.
-   **[Install all applications in a product](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-administration/application-manager/install-all-product-apps.md)**  
Install all applications and plugins included in a product using Application Manager.
