# Available templates

Use an application template to create an application in App Engine Studio \(AES\) with preconfigured data, experience, logic and automation, and security.

Templates are a good starting place for many applications you might want to build. You can also begin your application using a template and then customize the application. Use any of the templates below to build a fully-fledged application, or use them as a starting point to [[app-tutorial-create-an-app|create an app]] customized to your needs.

-   **[Document Approval template](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/app-engine-studio/document-approval-template.md)**  
Build an application in App Engine Studio \(AES\) using the [[document-approval-template|Document Approval template]] to manage the approval of documents in your organization.
-   **[Emergency Alert template](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/app-engine-studio/emergency-alert-template.md)**  
Build an application in App Engine Studio \(AES\) using the [[emergency-alert-template|Emergency Alert template]] to manage communications and track essential resources during an emergency.
-   **[Event Registration template](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/app-engine-studio/event-reg-template.md)**  
Build an application in App Engine Studio \(AES\) using the [[event-reg-template|Event Registration template]] to manage scheduling, booking, and registration for company events.
-   **[Expense Pre-Approval template](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/app-engine-studio/cost-approval-template.md)**  
Build an application in App Engine Studio \(AES\) using the [[cost-approval-template|Expense Pre-Approval template]] to create, track, and manage planned expenses in your organization.
-   **[Inventory Tracker template](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/app-engine-studio/inventory-tracker-template.md)**  
Build an application in App Engine Studio \(AES\) using the [[inventory-tracker-template|Inventory Tracker template]] to track the status of inventory and assets.
-   **[Performance Review template](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/app-engine-studio/performance-appraisal-template.md)**  
Build an application in App Engine Studio \(AES\) using the [[performance-appraisal-template|Performance Review template]] to automate scheduling, [[notifications|notifications]], and feedback for performance reviews.
-   **[Service Request Management template](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/app-engine-studio/service-request-management-template.md)**  
Build an application in App Engine Studio \(AES\) using the [[service-request-management-template|Service Request Management template]] to create, track, and manage service requests in your organization.
-   **[Team Contacts template](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/app-engine-studio/team-contacts-template.md)**  
Build an application in App Engine Studio \(AES\) using the [[team-contacts-template|Team Contacts template]] to manage contact information for the people in your organization.
-   **[Time Off template](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/app-engine-studio/time-off-requests-template.md)**  
Build an application in App Engine Studio \(AES\) using the Time Off Requests template to schedule employee vacations and time-off.
