# Transaction Manager personas

Personas in [[transaction-manager|Transaction Manager]] define the quoting interface and available actions for different user types in CPQ.

Personas are paired with views to control what [[fields|fields]] and events are visible to a user on the quote layout. Administrators create and configure personas in the Transaction Manager administration interface and assign them to the appropriate ServiceNow user roles.

|Persona|Description|ServiceNow role|
|-------|-----------|---------------|
|Sales agent|Creates and manages quotes from opportunities. Can add products, edit header fields, apply pricing, and submit for approval.| |
|Quote manager|Reviews and approves quotes. May have additional access to quote configuration or pricing fields.| |
